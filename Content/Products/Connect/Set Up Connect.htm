Set Up Connect

Overview

You can manage the majority of your Connect settings within simPRO. These settings apply to all Connect users.

To access your Connect settings in simPRO, go to System > Setup > Mobile, then adjust settings through the General, Pages, and Statuses sub-tabs as desired. You can also set up geostamping in simPRO to track where and when employees and, in simPRO Enterprise, contractors in the field update their mobile job statuses.

To manage how an individual Connect device retrieves, sends and displays data, access your In-app settings. To manage audits in Connect, learn more in Set Up Connect Audits.

General settings

Adjust general user permissions, runsheet settings and billing settings for Connect in simPRO. These settings will apply to all Connect users.

To adjust general and billing settings for Connect, go to System > Setup > Mobile > General. You can then restrict Connect functionality as desired with the following options:

Storage Device Access

This setting is separate from the security group permissions for assigning and transferring stock in simPRO. Select Do not restrict to give Connect users access to stock in all storage devices.

Select Restrict to assigned to restrict access to the assigned storage devices in their employee or, in simPRO Enterprise, contractor card files.
Today's Run Sheet Shows

Select Scheduled and due projects to show both scheduled and due jobs and quotes on the Today runsheet in Connect. See View the runsheet for more information.

Select Only scheduled to show only scheduled jobs and quotes. Jobs or quotes that are assigned but not scheduled will only appear in the Today runsheet when their due date is the same as the focus date.
Project Creation

Select Enable for jobs and quotes to allow users to Create a job in Connect and Create a quote in Connect (and new customers, sites, and if using simPRO Enterprise, leads). The created jobs or quotes will appear in simPRO after syncing.

Select Enable for jobs only or Enable quotes only as applicable.

Select Disable to prevent Connect users from creating jobs or quotes.
Travel Time

Select Schedule to project to schedule travel time to the project as a job cost.

Select Do not schedule if you wish for travel time to be non-billable.
Update Billable Labour

Select Automatically, based on schedule to have Connect update billable labour automatically when a job has been submitted.

Select Manually to have Connect users manually update billable labour.
Approve Jobcards

Select Automatically to automatically approve submitted job cards.

Select Manually to have job cards manually approved in simPRO.
Geostamping

Select Enabled to record the geographical location of a Connect user's mobile status updates, such as starting travel or starting a job.

Select Force to prevent users from switching off location services and allow additional time to acquire a geostamp.

See Geostamping settings for more information.

Billing settings

To adjust billing settings in Connect, go to System > Setup > Mobile > General.

Select Enable under Mobile Billing to allow field technicians to collect payments in the field.

Under the Payment Methods table, select Enable for Connect for the payment methods you want to enable field technicians to collect in the field. These can include simPRO Payments-associated card types.

 Pages settings

You can control which pages are visible to field technicians using Connect, as well as the order of pages in the workflow and jump menu. Disabling unnecessary pages reduces the number of steps in the workflow, making it faster for your field technicians to complete jobs.

To access the jump menu in Connect, access a job or quote from the runsheet, then tap the list icon in the top left.

Customise your workflow

To customise your Connect workflow in simPRO:

  1. Go to System > Setup > Mobile > Pages.
  2. Select Include in Work Flow for the pages you want your field technicians to navigate through when completing a job / quote.
  3. Select Include in Jump Menu for the pages you want your field technicians to be able to access through the jump menu.
  4. Click and drag the grey drag handle to move pages into the desired workflow order.
  5. Click Save.

You will need to Sync Connect with simPRO to update these changes.

If you are unsure about the right mobile job workflow for your business, we recommend starting with our standard workflow and reviewing enabled pages after using Connect. To view the workflow, see Connect Jobs.

You cannot remove the Details and Submit pages from the Connect workflow or jump menu.

Manage Attachments and Times

The Attachments and Times pages have additional settings.

To manage the Attachments page settings:

  1. Go to System > Setup > Mobile > Pages > Attachments.
  2. Click Settings.
  3. Select whether you wish to Make Uploaded Attachments Public.
  4. Select whether you wish to Allow Viewing of Private Attachments as defined in simPRO.
  5. Click Save.

To manage Times page settings:

  1. Go to System > Setup > Mobile > Pages > Times.
  2. Click Settings.
  3. Under Allow Employee to Specify Overtime, select Yes to let field technicians specify their own overtime in Connect.
  4. Enter the desired Time Block Threshold.
  5. Click Save.
 Statuses settings

Enable job and quote statuses for field technicians to update in Connect. You can use mobile status colours to make it easier for field technicians to see the status of all the jobs on their runsheet.

You can also set up these statuses to automatically update a job or quote in simPRO using triggers.

To set up statuses for Connect in simPRO:

  1. Go to System > Setup > Mobile > Statuses.
  2. Select Enable for the statuses you wish to use in Connect.
  3. To update the job or quote stage in simPRO when the mobile status is updated in Connect, select the desired job stage under Update to Job Stage.
  4. Use the slider to select colours to make the statuses easy to distinguish.
  5. Click Save to update your changes.

When you submit a job in Connect, the job will appear with a coloured strip on the runsheet, corresponding to the current status.

The mobile job statuses will also update the job status in simPRO to reflect the progress of the job according to the automatic triggers set up for Mobile Jobs.

It is recommended that you create status codes that are specific for mobile users, so you can easily distinguish if a job has been updated from a mobile device.

 Geostamping settings

Use geostamping for GPS-enabled Connect devices to know where and when your field technicians updated their mobile job and quote statuses.

To enable Connect geostamping in simPRO, go to System > Setup > Mobile > Geostamping and select Enable.

Select Force to prevent field technicians from progressing through the workflow or updating the job or quote status if their device's location services are disabled or unavailable. This also allows extra time to acquire a signal for the device.

It is not recommended to enable Force mode for Geostamping if users frequently operate in remote areas with poor signals.

View status updates

Once geostamping is enabled, you can view geostamped mobile status updates.

To view geostamped mobile status updates for several jobs:

  1. Go to Reports > View Reports > Mobile > Status Log.
  2. Filter the report as desired by selecting a Log Date and Employee(s).
  3. The coordinates recorded at the time of each mobile status update will appear in the Geostamping column.
  4. Select the check boxes in the Map column to show the locations of mobile status updates as pins on the map.

To view the mobile status updates for a specific job, go to the Mobile tab in a job card or the Schedule tab in a job.

The size of the circle on the map represents the accuracy of the geostamp, which can be affected by the strength of your network signal. You can adjust the status colour of the circle in your Statuses settings.

 In-app settings

Use in-app settings in Connect to manage what type of data your Connect device retrieves. You can also use the settings page to find information about your device's sync operations as well as the current version, server and user.

To access in-app settings, go to the Connect dashboard, then tap Settings. Alternatively, if you have not logged in to Connect, you can access in-app settings by tapping Settings on the login page.

Change the default language

You can change the default language in which you view Connect to proceed through the workflow more efficiently.

The current languages available in simPRO and Connect are:

  • English (Australia) / English (United Kingdom) / English (United States)
  • Spanish

You can change the default language that you view in Connect in the employee card in simPRO. Alternatively, set your default language in Connect by going to Settings > Language, then selecting the desired language from the drop-down list.

When you change the default language in Connect, your changes will reflect in simPRO, and vice versa.

The login screen will remain in English.

Manage data retrieval

To manage which jobs, quotes and tasks are available when you View the runsheet, you need to set up what data your Connect device retrieves from simPRO.

Under Data Retrieval Options, tap the drop-down list to select the job types you wish to retrieve from your simPRO server:

Pending and Progress View all outstanding assigned or scheduled jobs and quotes.
Pend, Prog, Comp, Inv. View all oustanding, completed and invoiced jobs and quotes that you have been assigned or scheduled to. Completed and invoiced jobs and quotes will appear in your Other runsheet.
All (+Archived) View all oustanding, completed, invoiced and archived jobs and quotes, you have been assigned or scheduled to regardless of date. Completed, invoiced and archived jobs and quotes will appear in your Other runsheet.

To view only outstanding assigned jobs and quotes, select Pending and Progress from the drop-down list, then select Also retrieve all pending/progress (Jobs / Quotes / Tasks) for other days. Your pending and progress jobs or quotes due or scheduled for dates other than the specified focus date will appear in your Other runsheet.

Select 48hr Scheduling to display all scheduled jobs or quotes within a 48-hour period of time when you View the runsheet for a 48-hour period, for example, if you have a night shift scheduled.

Select Use SSL to enable SSL attachment uploads. Clear if you wish to use HTTP attachments.

When you have finished setting up your data retrieval options, tap Finished and Connect will Sync in the background.

Manage sync operations

Manage your Sync Operations to monitor how Connect operates on your device.

Tap Go Offline, then Finished to use Connect without an internet connection. This option is useful when Connect cannot perform a sync due to an unreliable internet connection, for example, when a field technician is working on a job site in a remote area with poor mobile coverage.

When you use Connect in offline mode, all data will be stored on the device while you work in existing jobs and quotes. You cannot sync updated data to simPRO, request a purchase order, or create jobs, quotes or activities in offline mode. To allow the app to connect to the internet and sync with simPRO, tap Go Online, then Finished and Connect will automatically sync all updated data.

Tap View Log to view a log of app activity. If you are working with simPRO Support to resolve any issues you are having with Connect, you may need to send a copy of the log by tapping Post.

Tap Reset Database to delete all data from Connect and log out the current user. If you are working with simPRO Support to resolve issues with corrupt data, you may need to tap this option.

Various functionalities will be restricted when you Work offline in Connect.

View Connect information

You can view the following information under About simPRO Connect:

Version The current version of Connect used on the device. You need to access your device's app store to ensure you are using the latest version of Connect.
Server The simPRO build that the app is connected to and syncing with.
User The current user's name, or, if viewing Connect settings while logged out of the app, the name of the last user to have logged in.
Last Sync A timestamp of the most recently recorded sync between Connect and simPRO.
Uploaded Status Indicates whether there is data that needs to be synced with simPRO.
Device The mobile device type and operating system.
