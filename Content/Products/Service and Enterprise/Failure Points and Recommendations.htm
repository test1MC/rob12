Failure Points and Recommendations (Maintenance Planner only)

Overview

Use failure points as specific test points for your technicians to use when performing maintenance on customer assets. You can also add recommendations so that you can choose what to do next if an asset fails a test.

Add failure points

You will need to add failure points to each service level on an asset. You can also add a recommendation for each failure point.

To add failure points:

  1. Go to System > Setup > System Setup > Customer Assets > Asset Builder.
  2. Click the desired asset.
  3. Click Service Levels.
  4. Click Options > Edit Failure Points on the desired service level.
  5. Click New.
  6. Enter the name and Standard ID of the failure point.
  7. Select Critical? if applicable.
    • This will create a more prominent alert if the failure point is selected for an asset.
  8. Select a set-price Pre-Build.
    • If this failure point is selected when the asset is failed and you create a quote for it, this pre-build will automatically be added.
  9. To add a recommendation, click the arrow in the left-most column, then click New.
    • Enter the recommendation Name.
    • Select a Pre-Build that will be automatically added to the quote if the technician selects this recommendation.
    • Enter the Charge Rate. Alternatively, leave this blank to use the default price.
  10. Continue adding failure points / recommendations as desired.
    • Technicians will be able to select a recommendation after an asset fails testing.
  11. Click Finish.
 View failed assets

You will receive an alert if an asset fails a test. You can then view the failure points that were selected and decide how to proceed.
