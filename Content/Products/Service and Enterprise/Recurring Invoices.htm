Recurring Invoices

Overview

Use recurring invoices to create a regular financial record independent of any jobs in the simPRO database. This is useful if you provide a regular billable service to clients over a period of time, such as security monitoring, or if the frequency of planned maintenance doesn't coincide with the frequency of invoicing.

Create a recurring invoice template

Instead of creating an invoice manually each time, create a recurring invoice template. Add parts and labour and set the invoice template frequency to receive an alert to create the invoice for the customer at the defined regular intervals.

You can create recurring templates for both service and project invoices.

You can create recurring templates for both service and project invoices.

If you want to make a change to the recurring invoice, you need to edit the template itself and not just one occurrence of the invoice. Only changes made to the template will affect invoices created in the future.

Create a recurring service invoice template

To create a recurring service invoice template:

  1. Go to Invoices > Create Recurring Invoice.
  2. Search and select the desired Customer or customer contract.
  3. Select the Site.
  4. Enter the Order Number and Request Number as applicable.
  5. Select Service as the Recurring Invoice Type.
  6. Select Service as the Recurring Invoice Type.
  7. Select a Cost Centre and enter a Description.
  8. Select the Recurring Start Date, then set the Recurring Frequency by selecting the number of Week(s), Month(s), or Year(s) you wish to have the invoice recur.
  9. If desired, assign a Technician, Salesperson and / or Project Manager to the invoice.
  10. If desired, assign a Technician, Salesperson and / or Project Manager to the invoice.
  11. If desired, assign a Technician or Salesperson to the invoice.
  12. Click Next.
  13. Click Parts & Labour to add billable items and labour as you would for a service job.
  14. Click Details to review your Summary.
  15. Click Finish.
 Create a recurring project invoice template

To create a recurring project invoice template:

  1. Go to Invoices > Create Recurring Invoice.
  2. Search and select the desired Customer or customer contract.
  3. Select the Site.
  4. Enter the Order Number and Request Number as applicable.
  5. Select Project as the Recurring Invoice Type.
  6. Select the Recurring Start Date, then set the Recurring Frequency by selecting the number of Week(s), Month(s), or years you wish to have the invoice recur.
  7. If desired, assign a Technician, Salesperson and / or Project Manager to the invoice.
  8. Click Next.
  9. Add Sections, Cost Centres, and parts and labour as you would for a project job.
  10. Click Save.
  11. Click the Recurring Invoice # in the breadcrumbs to review your Summary.
  12. Click Finish.

 Create an invoice from a template

Once you have created a recurring invoice template, you will receive an alert whenever an invoice is due.

To create invoices from all recurring invoice templates that are due:

  1. Go to Invoices > Recurring Invoices.
  2. Select Select All Due.
  3. Click Create Invoices.

You can then locate, update, and email the newly created invoice / s in Invoices > Unpaid Invoices.

 Skip a period

You can skip a period of a recurring invoice so you do not receive an alert until the following due date.

To skip a period for an individual recurring invoice template, go to Invoices > Recurring Invoices, then click Options > Skip Period on the desired recurring invoice template.

To skip a period on multiple recurring invoices:

  1. Go to Invoices > Recurring Invoices.
  2. Select the check boxes for each template you wish to skip.
  3. Click Skip Period at the bottom of the page.
  4. The Next Due Date column will then display the updated next due date.

It is recommended to add a note to the recurring invoice template when a period is skipped to keep record of when an invoice was not created.
