Prepaid Labour Types

Overview

If you sell prepaid or up-front services to customers, you can use prepaid time in simPRO.

To set this up, you will first need to create prepaid labour types and prepaid blocks, which you can use to sell prepaid time to a customer.

If you have customers with multiple sites, and your customer needs to control which sites have access to prepaid time, you may wish to first enable customer and site based prepaid time.

Create a prepaid labour type

To create a prepaid labour type:

  1. Go to System > Setup > System Setup > Labour > Prepaid Labour Types.
  2. Select a Labour Name. These are your existing labour rates.
  3. If desired, enter a Multiplier.
  4. Click Create.
 Edit a prepaid labour type

To edit a prepaid labour type:

  1. Go to System > Setup > System Setup > Labour > Prepaid Labour Types.
  2. Click the prepaid labour type you wish to edit.
  3. Make changes as desired.
  4. Click Save.
 Delete a prepaid labour type

To delete a prepaid labour type:

  1. Go to System > Setup > System Setup > Labour > Prepaid Labour Types.
  2. On the prepaid labour type you wish to delete, click the red delete icon.
  3. Click OK.
 Advanced

Enable customer and site based prepaid time

If you have customers with multiple sites, and your customer needs to control which sites have access to prepaid time, you may wish to first enable customer and site based prepaid time. When you are ready to invoice a completed job, prepaid time will only be available on invoices being raised for a site which has already purchased prepaid time.

To enable customer and site based prepaid time:

  1. Go to System > Setup > Defaults > System > General > Prepaid Time.
  2. Select Customer & Site based.
  3. Click Save.
