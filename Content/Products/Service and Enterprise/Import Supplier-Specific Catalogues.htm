Import Supplier-Specific Catalogues

Overview

Upload supplier catalogues and price files that have been specifically formatted for your simPRO system.

Catalogues are usually contained in a CSV or ZIP file. To obtain these files, you need to contact suppliers directly at your local branch or log in to their website. If the supplier is not aware of the simPRO format required they should contact their internal IT department for support.

There are three types of imports you can perform with a supplier-specific catalogue:

Complete

Imports all items from the price file as arranged in the groups and subgroups defined in the file. Select when performing an initial import.

Update and Create New Part

Compares the part numbers of items from the price file with those in the existing catalogue. If a part number in the new price file is not found in the catalogue, a new item entry will be created in the catalogue in a group and subgroup defined by the new price file.

If you are updating your catalogue with your default supplier, trade prices will also be updated. This option is useful if you wish to build up your catalogue to include all products from additional suppliers.

Update Only

Compares the part numbers of items from the price file with those in the existing catalogue but ignores additional items in the new price file. If you are updating your catalogue with your default supplier, trade prices will also be updated.

If you are updating your catalogue using the Update and Create New Part or Update Only options and matching part numbers are found between the catalogue and the new price file, the import will merge those items into one catalogue item with multiple suppliers. This is useful for comparing prices between different suppliers when raising a purchase order.

If you are unable to obtain a simPRO formatted price file from your supplier, you may wish to import the supplier catalogue as a generic price file or manually enter your most commonly used catalogue items.

Import a supplier-specific catalogue

Before you import a supplier-specific catalogue, you need to Import Suppliers, or alternatively, create Suppliers.

Perform an initial import

When performing an initial import of a supplier-specific catalogue, it is recommended to import the catalogue of your primary supplier, or alternatively, a major supplier with a well-structured catalogue.

To perform an initial import of a supplier-specific catalogue:

  1. Go to Utilities > Import.
  2. In the left column, hover over Catalogue, then select the supplier whose catalogue you wish to import.
  3. Select Set as Default Supplier if desired.
  4. Under Import Options, click Complete.
  5. Click Browse.
  6. Find and click the desired supplied file, then click Open.
  7. Under Attached File, a green tick icon will indicate the file has uploaded successfully. Verify the name and size of the file.
  8. Click Next.

When the catalogue has finished importing, an Import Catalogue Summary will show the number of Line Items Loaded, Catalogue Items Created and Catalogue Items Updated.

It is recommended that you check the catalogue to ensure the import has been successful prior to importing additional supplier catalogues.

 Update the catalogue

Update your existing catalogue to import additional items from any supplier, update items with additional supplier part numbers and prices, or simply update trade prices.

If you are updating the catalogue with item details from MMEM, you can check for a new price file and perform an automatic sync.

To update the catalogue:

  1. Go to Utilities > Import.
  2. In the left column, hover over Catalogue, then select the supplier whose catalogue you wish to import.
  3. Select Update Trade Prices if desired when updating the catalogue for your default supplier.
  4. Under Import Options:
  5. Click Update Only to update existing items with supplier part numbers and prices.
  6. ​​Click Update and Create New Part to update existing items with supplier part numbers and prices, and to add additional items.
  7. Click Browse.
  8. Find and click the desired supplied file, then click Open.
  9. Under Attached File, a green tick icon will indicate the file has uploaded successfully. Verify that the correct file has been loaded.
  10. Click Next.

When the catalogue has finished importing, an Import Catalogue Summary will show the number of Line Items Loaded, Catalogue Items Created and Catalogue Items Updated.

Supplier requirements

If you wish to import from Espro, a database of electrical contracting pre-builds, you can import a sample of 200 items. To import the full database, you need to purchase it from Spearhead, then provide proof of purchase to simPRO Support, and simPRO will load the database to your system.

To import the Middy's price file directly from the Middy's website into simPRO, adjust import options, enter your Middys User Name and Middys Password, then click Next.

 Enable automatic syncing for MMEM

If you import MMEM supplier catalogues into your simPRO system and you have an online MMEM account, you can enable automatic syncing so that your prices will update in simPRO when MMEM updates their price files.

To enable automatic syncing for the MMEM catalogue:

  1. Go to Utilities > Import.
  2. In the left column, hover over Catalogue, then select MMEM.
  3. Enter your MMEM Username and Password.
  4. Click Validate.
  5. Adjust import settings as normal.
    • If you wish to have different import settings for future updates, return to Utilities > Import > Catalogue > MMEM > Import Options, select options as desired, then click Update.
  6. Click Finish.

Your simPRO system will then check for a new MMEM price file every seven days and will update in the system regardless of whether the price file has changed. To disable automatic syncing, click Deactivate.

Go to Utilities > Import > MMEM > File Import, to import an MMEM supplier catalogue manually.

Do you have an invalid MMEM username or password?

If you have an invalid MMEM username or password, a Catalogue Automated Import Error message will appear. Go to the MMEM website to reset your password, or alternatively, contact your local MMEM to rectify the issue if required.
