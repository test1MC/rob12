Navigation and Data Management Tools

Overview

Throughout simPRO, you'll find some tools common across lots of areas that will help you navigate the system and manage your data.

Navigate simPRO

Click breadcrumbs

Breadcrumbs are displayed below the main menu on all pages except the dashboard. The right-most label is your current location in the system.

Click the breadcrumbs to the left to return to previous pages instead of using your browser's back button.

Perform a global search

Use the global search bar to the right of your simPRO interface in the main menu to search for actions, records, and card files throughout your entire simPRO system.

To perform a global search, enter key terms, such as the customer or site name, then select from the drop-down list as applicable. The drop-down list displays the first ten results for your search and will only appear if you do not click Search.

You can also hover over the bookmark icon to view and access your saved searches.

To perform a global search and view all results, enter the desired search terms, then click Search. You can then view results by Category, as well as the number of Results within Category.

When you click a Category, you will be taken to the applicable data table with the results automatically filtered according to the Advanced local search filters. You can then save the search by clicking Save Search.

 Multi-company: Company Selector

If you have a Multi-company build, click the tab displaying the name and colour of the company you are currently using on the left side of the simPRO interface to view the Company Selector. You can then click another company to access it.

The Company Selector is available everywhere in simPRO except when you are creating or editing a data entry.

 Save your changes in simPRO

When you create or edit a data entry in simPRO, such as a job, quote, customer, employee, supplier, invoice, and more, there are several options for saving your data.

Click Cancel to exit the data entry without saving. If you have just created the data entry, no record of it will save in simPRO unless you have already clicked Save.

Click Save to back up your data entry in simPRO and continue editing. If you have just created the data entry, clicking Save will create a record number or ID for the entry and ensure access from its applicable data table.

Click Finish to save and exit the data entry. You may then need to select from various options for what to do next. For example, in a quote, you will have the options to view quotes, convert the quote to a job, edit the quote or schedule.

If you are editing a cost centre in a project job / quote or service job / quote with multiple cost centres, click Continue to save and return to the Cost Centre List.

If you are editing a cost centre in a project job / quote or service job / quote with multiple cost centres, click Continue to save and return to the Cost Centre List.

 Access options

Click Options on an entry in a data table to view available actions. You can also right-click anywhere on the line entry to view the same options.

When viewing data tables for leads, quotes, jobs, invoices, contractor work orders or purchase orders, click '...' next to Options to view related data entries. For example, a job that has been invoiced will have the invoice listed underneath it.

 Perform a local search

Perform a local search as a quicker and more precise alternative to the global search bar by searching for data entries within a specific data table.

To perform a local search, access the desired data table, for example, People > Customers, then enter search terms as applicable and click Search. The data table will then display only data entries relating to the entered search terms.

To narrow your search results, click Advanced, then select from the filters as applicable. To hide Advanced options, click Basic. If you still have Advanced search filters applied to a data table, a yellow circle with a number inside will appear next to Advanced to indicate how many filters are applied.

To remove the applied search filters or search terms, click the grey x icon in the local search bar.

 Save searches

If you perform a particular search often, make it a saved search so you can quickly perform it from the global search bar, which is located underneath the utilities menu.

To save a search:

  1. Click Save Search after searching.
  2. Enter the Saved Search Name.
  3. Select whether to make this saved search available to Only Me or All Employees.
  4. Click Save.

To view your saved search, hover over the bookmark icon next to the global search bar.

To remove a saved search, click the red delete icon.

 Modify the data table view

Click Modify Table View to adjust which information fields appear in a data table and the order they appear in. Select the check boxes for the fields you wish to include, and clear the check boxes for the fields you wish to exclude.

You can also Reorder fields by clicking and dragging them into the desired order. Click Finish to save your changes and view your modified table.

Using Modify Table View will change the table view for all users. It is recommended that you use security groups to restrict modifying table view to only certain employees.

 Reorder data

In some data tables, such as the Catalogue, Pre-Builds, Take Off Templates, and Tasks, you can change the order in which data entries appear.

To reorder an entry, click and drag the grey bar on the left to the desired position.

 Apply footer options to multiple data entries

Use the footer options below a data table to apply settings or perform actions for multiple data entries at once, for example, when you wish to archive entries in bulk.

Select the check boxes for the desired data entries, then click the desired action in the footer options. These actions vary for different data tables and are usually similar to the actions available in Options.

If you are changing settings for multiple data entries at once, you can also select from the drop-down lists in the footer options, then click Update.

You can also Select All to apply footer options to all data entries on the current page.

 Open simPRO in a new browser tab / window

In addition to having multiple projects open at the same time, you can also open simPRO links, such as a hyperlinked job number or a menu option, in a new browser tab or window.

Hold the Ctrl key on your keyboard, then click the desired link to open a new browser tab. Alternatively, right-mouse click the desired link, then select to open it in a new browser tab or window as desired.
