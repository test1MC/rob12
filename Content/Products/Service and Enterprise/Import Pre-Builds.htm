Import Pre-Builds

Overview

Pre-builds combine materials and labour as a single billable item to appear on jobs, quotes and invoices. You can manually create pre-builds in simPRO, or alternatively, you can save time and reduce data entry errors by importing a pre-build list.

You may wish to test the import process with one or two pre-builds before importing an entire list.

Prepare to import pre-builds

To prepare to import pre-builds into simPRO:

  1. Download, add data to, and edit the Pre-Build Import Template.
  2. Download, add data to, and edit the Pre-Build Import Template.
  3. Download, add data to, and edit the Pre-Build Import Template.
  4. Download, add data to, and edit the Pre-Build Import Template.
  5. Download, add data to, and edit the Pre-Build Import Template for:
    • Australia
    • New Zealand
    • United Kingdom
    • United States
  6. Alternatively, you can create your own import list and leave out any fields that do not apply.
  7. Separate your data using the same or similar headings as the Pre-Build Import Template.
  8. Leave out any fields that do not apply to your pre-builds.
  11. Save your file in your system as a compatible file type: CSV (.csv like the Pre-Build Import Template above), text (.txt) or a ZIP file (.zip). Consult your spreadsheet software for file saving instructions if required.

simPRO can recognise and import all the headings and information listed in the Pre-Build Import Template. If you entered data in the Pre-build Price column, the pre-build will be added as a set price pre-build.

Columns for Group, Part No. and Pre-build Name are required for importing. The Group and Pre-build Name columns must also contain data; this is not necessary for the Part No. column.

If a pre-build has a numeric-only custom field in simPRO but there is non-numeric data entered in the field in the pre-build list, you will receive an error warning and the import will fail. Numbers must be formatted as specified in System > Setup > Defaults > System > Number Thousands Separator / Negative Number Format to import successfully.

Import a pre-build list

To import a pre-build list:

  1. Go to Utilities > Import > Pre-Builds.
  2. Click Choose File / Browse.
  3. Locate your pre-build import file and click Open.
  4. Click Next.
  5. Match your pre-build list fields to the fields in simPRO using the drop-down lists.
    • To leave a column out of the import, select Ignore.
  6. Click Finish.

You will be taken to the Pre-Build Import Summary Page. It is recommended to review your imported pre-builds to check for errors by going to Materials > Pre-Builds.

 Amend pre-build import errors

If your pre-build import file does not contain the correct data or columns, your pre-builds will not import and you will receive the Errors found in Import File error message.

To amend this error, you will need to edit the relevant data or columns as appropriate and import the pre-build list again as normal.

If you receive the Errors found during processing of Pre-builds message, the pre-builds have uploaded into simPRO, but you may need to access the relevant pre-build and / or part and make changes as necessary.
