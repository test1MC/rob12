eForms Form Designer

Overview

Create and manage forms in System > Setup > Forms in the eForms portal.

To create and edit forms, you need to have access to the Form Editor and Form Designer enabled in your security group settings.

Manage forms

To manage your forms in the eForms portal, go to System > Setup > Forms. You can then view all form templates created along with their Form Name, unique Form ID, Version as well as who has created / edited the template and when.

To filter and locate specific form templates, enter the Form Name or select Show Generic Forms Only / Show Archived Forms Only to view generic / archived forms.

Click a form template to edit, or alternatively, click Options > Edit Form. You can also copy and archive forms using the Options button.

 Create a form template

To create a form template:

  1. Go to System > Setup > Forms.
  2. Click Create Form.
  3. Enter an easily identifiable Form Name, then click Next.
  4. Read the terms and conditions of legal indemnity, then select I Agree if applicable.
  5. Click Upload Background, then locate and open the desired file.
    • JPEG, PNG, GIF, PDF - image resolution: minimum width of 850px required
  6. If applicable, select Form Requires Review to prevent the form being processed until a review is formed in the eForms Portal.
  7. Adjust Form Settings as desired.
  8. Under Form Tools, select a specific field type, then click on the form background where you wish to place it.
  9. Enter an Element Name, adjust the field editing options as desired, then click Save.
    • Click and drag the resize icon in the bottom right corner of an element to resize it.
    • Click Unlock to reposition the field using the white drag icon.
    • Click Clone to copy the field.
    • Click Delete to delete the field.
  10. Click Save as you complete each element.
  11. Click Finish to save and complete the form and return to the eForms Portal.

It is recommended to test the layout and field sizes in forms created in the eForms Designer by creating a practice form and accessing it on a tablet.

Adjust form settings

When you create a form template, you can adjust form settings so that all forms completed in that template must be reviewed in the eForms Portal before being processed. You can also select and enter email addresses to distribute the form to once completed.

To adjust form settings:

  1. Create or access a form as normal, then go to Settings > Form Settings.
  2. Select whether the Form Requires Review as desired.
  3. Select Email to Submitter to have a copy of any form completed in the form template emailed to the submitting technician.
  4. To select or enter email address to distribute the form to once completed, select Email Form and click:
    • Show Users to view all licensed eForms users. Click a name to add their email address to the Selected Mail Recipients list. Click the X icon to remove an email address.
    • Add Email to enter a new Email Address, then click Add. Click the X next to an email address to remove it.
  5. Make changes to the form as desired, then click Finish.
 Use form tools and editing options

Under Form Tools, click to add specific field types to a form, using the editing options to adjust how information can be entered.

In each field type, you need to enter an Element Name to identify the type of information to be entered in the field, for example, Customer Name or Address. The Element Name must contain only alphanumeric characters, square brackets and underscores, with no punctuation or spaces allowed.

If you have enabled simPRO integration, the Element Name will populate with the pre-assigned element selected.

The following field types and editing options are available in the eForms Designer:

simPRO Integration

Use to allow areas to be auto-populated with simPRO data such as the Job Number or Quote Number if you have connected with simPRO through System > Setup > Extensions. Edit the Font Size and Font Format of the text entered in the field, or select to Hide from PDF if desired.

Text Field

Use for short, one line text inputs such as the customer name. Edit the Font Size and Font Format of the text entered in the field.

Text Area

Use for paragraph text inputs, such as addresses, descriptions or notes. Edit the Font Size and Font Format of the text entered in the field.

Signature Field

Use to allow a technician or customer to sign an area within the form.

Drawing Panel

Use to allow a technician to draw a plan or location for the work required.

Radio Button

Use for list items that require only one response. Enter a Field Value, that is, the name of the option, and whether the option should be Automatically checked.

Tick Box

Use for list items that can have multiple responses. Enter a Field Value, that is, the name of the option, and whether the option should be Automatically checked.

Date Selector

Use for date fields. Edit the Font Size and Font Format and select the Display current Date? check box if desired.

Time Selector

Use for time fields. Edit the Font Size and Font Format and select the Display current Time? check box if desired.

Select Field

Use to display list items in a drop-down list for single or multiple selection. Edit the Font Size and Font Format, and enter the applicable list options in the Option Text / Option Value fields, adding or deleting options as necessary. Select Multiple Select? to allow multiple selection of options.

Image Upload

Use to allow a technician to capture an image and embed it into a form. Edit the width and height of the Element Size.

It is recommended to test the layout and field sizes in forms created in the eForms Designer by creating a practice form and accessing it on a tablet.
