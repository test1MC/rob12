Task Templates and Categories

Overview

Task templates and categories can be created, edited and deleted through System > Setup > System Setup > Tasks.

If you frequently carry out the same set of tasks for certain types of quotes or jobs, create a task template containing each task to quickly create and add those tasks to the quote or job. Once created, adding a task template to a quote or job will create all of its individual tasks, which you can then manage individually. Templates are particularly useful for project work.

Task categories can be used to easily group, filter and bulk-manage tasks. Create task categories to assign them to tasks.

Create a task template

To create a task template:

  1. Go to System > Setup > System Setup > Tasks > Task Templates.
  2. Click Create Task Template.
  3. Enter a Name.
  4. Click Add Row.
  5. Enter the task name in Subject.
  6. Enter additional task details in the Description field as required.
  7. If desired, drag and drop attachments.
  8. Create additional rows for each task.
  9. Click Create.
 Edit a task template

To edit a task template:

  1. Go to System > Setup > System Setup > Tasks > Task Templates.
  2. Click the template you wish to edit.
  3. Make changes as desired.
  4. Click Save.

Changes to task templates will not affect tasks already created using that template.

 Delete a task template

Deleting a task template will not delete any tasks it has been used to create.

To delete a task template:

  1. Go to System > Setup > System Setup > Tasks > Task Templates.
  2. On the task template you wish to delete, click the red delete icon.
  3. Click OK.
 Use a task template in a quote or job

To use a task template in a quote or job:

  1. Open the quote or job, or create a new one.
  2. Go to Details > Tasks.
  3. Under Task Templates, select a template from the Template drop-down list.
  4. Click Add Tasks.
  5. Review the newly created tasks under Tasks.

The new tasks will be available through the quote or job, and through Tasks > View on the dashboard.

 Create a task category

To create a task category:

  1. Go to System > Setup > System Setup > Tasks > Task Categories.
  2. Click Create Task Category.
  3. Enter a Name.
  4. Click Create.
 Edit a task category

To edit a task category:

  1. Go to System > Setup > System Setup > Tasks > Task Categories.
  2. Click the category you wish to edit.
  3. Make changes as desired.
  4. Click Save.
 Delete a task category

Deleting a task category will remove it from all tasks it is assigned to.

To delete a single task category:

  1. Go to System > Setup > System Setup > Tasks > Task Categories.
  2. On the task category you wish to delete, click the red delete icon.
  3. Click OK.

To delete multiple task categories at once:

  1. Go to System > Setup > System Setup > Tasks > Task Categories.
  2. Select the check boxes for the categories you wish to delete, or click Select All.
  3. Click Delete.
  4. Click OK.
 Assign a task category

To assign a category to a task from within the task:

  1. Open the task, or create a new one.
  2. Go to Details > Internals.
  3. Select a category from the Category drop-down list.
  4. Click Save or Finish.

To assign a category to one or more tasks from the Tasks page or from within a quote or job:

  1. Go to Tasks > View, or if viewing a quote or job go to Details > Tasks.
  2. If desired, click Advanced to filter tasks as required.
  3. Select the check boxes of the desired tasks.
  4. At the bottom of the page, under With selected, select a category from the Category drop-down list.
  5. Click Update.
 Filter tasks by category

Filter tasks by category to narrow down the tasks shown in Tasks > View. For example, you may wish to view the status of tasks in a certain category. After filtering, you can bulk-manage the tasks shown as needed, such as reassigning all tasks in a certain category to someone else.

To use categories to filter tasks:

  1. Go to Tasks > View.
  2. Next to the search bar, click Advanced.
  3. Under Filter By, select a category from the Category drop-down list.

The list of tasks will update to only show tasks from the selected category.
