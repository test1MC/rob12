CIS Deductions Report (UK only)

Overview

You can use the CIS Deductions report to view CIS deducted by each customer in a specified date range.

The report can provide a simple summary or detailed information on dates paid, amounts, materials, labour and CIS deducted.

This article only provides instructions for using the CIS features in simPRO, and should not be interpreted as legal or financial advice. Read more about CIS in our CIS Guide (UK Only), or read about your CIS obligations on the UK Government's CIS website.

View the report

To view the CIS Deductions report:

  1. Go to Reports > View Reports > Sales > CIS Deductions.
  2. Use the date picker to select a Date Range.
  3. Select whether you want to view the report Type as a Summary or a Breakdown.
  4. If desired, select a Customer.
  5. Click Search.

​If you select the report Type as Summary, you will simply see the CIS Deducted for each customer in the selected date range.

If you select the report Type as Breakdown, results for the selected date range will be grouped by customer. For each invoice you will see Date Paid (if applicable), Amount (Ex Tax), Materials, Labour, and CIS Deducted.
