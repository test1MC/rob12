Import Customer Notes

Overview

Use customer notes in simPRO to record any correspondence logged against a customer. You can manually enter notes about customers in the Notes tab of a customer card file, or save time and reduce data entry error by importing a customer notes list.

If you include a Notes column when importing a customer list, the data will upload to the Notes text box in Profile > Settings in the customer card file.

Customers for whom you wish to import notes for must exist in simPRO before importing.

Prepare to import a customer notes list

To prepare to import customer notes into simPRO:

  1. Ensure the corresponding customers exist in simPRO. You can manually create or import customers.
  2. Save your file in your system as a compatible file type: CSV (.csv), text (.txt) or a ZIP file (.zip). Consult your spreadsheet software for file saving instructions if required.
  3. Separate your data using clear headings.
    • To import successfully, specify the headings for Note, First Name and Last Name for an individuals, and Company Name for companies.
    • Other headings may include Subject and Date.

Notes data including Reference No.Follow Up and Assign To must be entered manually in the customer card file.

 Import a customer notes list

To import a customer notes list:

  1. Go to Utilities > Import > Customers > Customer Notes.
  2. Under Import Options - First Record is, select:
    • Header Record if the first row of your customer notes list contains column titles.
    • Data Record if the first row contains specific data.
  3. Under Customer Status, select whether the upload file contains Customers or Leads.
  4. Under Customer Status, select whether the upload file contains Customers or Leads.
  5. Click Choose File / Browse to locate and open your customer notes import file.
  6. When the name of the file displays, click Next.
  7. Match Customer Notes Import Columns as applicable using the drop-down lists.
  8. If you do not want to include a column in the import, select Ignore.
  9. Click Import Customers.

You may wish to test the import process with one or two customer note items before importing an entire list.

Check import success

After uploading an import file, the Import Customer Notes Summary page will display a breakdown of your import, including Notes Added and Customers Multi Matched.

Click Customers Notes imported successfully to download a CSV file of imported customer notes for your records.

After a successful import, it is recommended you manually check customer card files to ensure customer note data has imported correctly. To do this, go to People > Customers, click an imported customer, then check the Notes tab for errors. Click a note to manually amend if required.

 Amend import errors

If your customer notes import file does not contain columns for Note, First and Last Name or Company Name, the import will fail and you will receive an error message.

To amend the error, click OK, then enter details in the required fields in your import list and restart the import process.

If your file contains other errors, an error message will display on the Import Customer Notes Summary page, and the table will display the number of errors found.

Click click here to download a CSV file of import errors. The Error Text column will explain why entries failed. Amend your customer notes import list accordingly and restart the import process.
