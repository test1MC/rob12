Import Plant and Equipment

Overview

You can manually Create Plant Items individually, or you can import a plant item list.

You will need to have the correct security group permissions to import a plant list.

You may wish to test the import process with a short list before importing all of your plant items.

Prepare to import

To prepare to import plant items into simPRO:

You will need to create a separate import file for each of your plant types.

  1. Download the simPRO Plant Import Template:
    • ​​Plant Import Template - Service
    • Plant Import Template - Enterprise
  2. Add any custom fields for the particular plant type as additional columns. Ensure that the headings of your columns match the spelling of your custom fields in simPRO.
  5. Save your file in your system as a compatible file type: Comma Separated Values (.csv like the Plant Import Template above), Tab Delimited, also known as a text file (.txt), or a ZIP file (.zip). Consult your spreadsheet software for file saving instructions if required.

simPRO can recognise and import all the headings and information listed in the template.

Field requirements

You can import the following fields of information when uploading plant lists to simPRO:

Plant ID

If you are using the import to update your plant items, the Plant ID must be identical to how it is defined in simPRO. When you open a plant item, the plant ID is the number at the end of the URL.

To create a new plant item, leave this column blank.

Name

The name of the plant item. This is a mandatory field and must be completed to ensure the creation and updating of plant items.

Servicing

Enter TRUE if you wish to service the item, or FALSE if not.

Billing

Enter TRUE if you wish to invoice customers for use of the item, or FALSE if not.

Service Start

This field is based on the plant type settings and is applicable if the item requires regular servicing. If the plant type is serviced based on Date, dates must be formatted as specified in System > Setup > Defaults > System > Date Format to import successfully.. If the plant type is serviced based on Hours, then this must be a value between 0 and 999.

Service Frequency

This field is based on the plant type settings and is applicable if the item requires regular servicing. If the plant type is serviced based on Hours, then this must be a value between 0 and 999.

Service Frequency Period

This field is applicable if the item needs regular servicing and if the plant type is serviced based on Date. This can be either 'week', 'month' or 'year'. If the plant type is serviced based on Hours, this column can be ignored.

Cost Rate

This field is only applicable if the item is marked as billable and is the hourly cost of the plant item.

Overhead Rate

This field is only applicable if the item is marked as billable and should be the same overhead used with your labour rates.

Sell Rate

This field is only applicable if the item is marked as billable and is the hourly rate of the plant item when billing customers.

Sales Tax Code

This field is only applicable if the item is marked as billable and is the tax code used when invoicing the plant item. Tax codes must be created in simPRO prior to importing.

If your plant type has custom fields, you can include information defined by custom fields specific to the plant type.

To apply custom field information to your plant list:

  1. Locate and open your plant list.
  2. In simPRO, go to System > Setup > System Setup > Plant Builder.
  3. Click the relevant plant type as defined in the list.
  4. Click the Custom Fields tab.
  5. For each custom field, add a new column to the plant list, heading each column with the relevant Field Name.
  6. The Field Name headers must be identical to how they are formatted in simPRO. It is recommended to copy and paste field names to ensure a successful import.
  7. Enter data in the columns as required.
  8. Similar to the Field Name, custom field data in the form of list options must be identical to how it is formatted in simPRO. If you have entered new list options in your import list, select Create New List Option when importing.
  9. When you have finished entering custom field data for that plant type and list, save your file and repeat the process with your other plant lists.

Import a plant item list

To import a plant item list:

  1. Go to Utilities > Import > Plant & Equipment.
  2. Under First Record Is:
    • Select Header Record if the first row of your file is column headings.
    • Select Data Record if the first row of your file is data.
  3. Under If Custom List Value Not Found:
    • Select Create New List Option to include custom list values in the import that do not exist in simPRO, such as a list option in a custom field.
    • Select Reject and Warn to exclude new custom list values from the import.
  4. Select Plant Type as applicable.
  5. Click Choose File / Browse.
  6. Locate your plant item import file and click Open.
  7. Click Next.
  8. Match your plant item list fields to the fields in simPRO using the drop-down lists.
    • To exclude a field from the import, select Ignore.
  9. Click Finish.

If the import is successful, you will be taken to the Import Plant & Equipment Summary page. It is recommended to review your imported plant items to check for errors by going to Utilities > Plant & Equipment.

To amend import errors, access the relevant plant items in simPRO individually and manually make changes as desired. Alternatively, adjust your import list as required, then re-import the plant items with their newly-created plant IDs.
