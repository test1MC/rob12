Import Suppliers

Overview

In simPRO, you can manually add suppliers to the system through People > Create New Supplier, or save time and reduce data entry errors by importing a supplier list.

You may wish to test the import process with one or two suppliers before importing an entire list.

  7. Save your file in your system as a compatible file type: CSV, .txt, or .zip. Consult your spreadsheet software for file saving instructions if required.
  8. Separate your data using the same or similar headings as the Supplier Import Template.

If a supplier has a numeric-only custom field in simPRO but there is non-numeric data entered in the field in the supplier list, you will receive an error warning and the import will fail. Numbers must be formatted as specified in System > Setup > Defaults > System > Number Thousands Separator / Negative Number Format to import successfully.

Import a supplier list

After preparing your supplier list, work through the following steps to import your list into simPRO.

Adjust supplier import settings

To adjust supplier import settings, go to Utilities > Import > Suppliers.

The options available include:

First Record Is

: Select Header Record if your list's first row contains column titles, or select Data Record if your list's first row contains specific supplier details.
If supplier Already Exists

In case simPRO finds duplicated suppliers (supplier with the same name), select whether you want to Do Nothing or to Update Supplier Details for the existing supplier in simPRO.
If Custom List Value Not Found

If any of the entries are a drop-down list in simPRO (for instance, the State field is a drop-down list), and your file contains any additional values (or 'custom list values') not available in simPRO's list, select Create New List Option to add the additional value to the drop-down list in your supplier card file. Otherwise, select Reject and Warn to ignore the field and receive a warning.

Select an import file

To select an import file:

  1. Go to Utilities > Import > Supplier.
  2. Adjust supplier import settings, then click Choose File / Browse.
  3. Locate your supplier import file and click Open.
  4. Click Next.

Match supplier import columns

After selecting an import file, manually match or check your list columns with the fields available in simPRO. You may wish to have both simPRO and your list open at the same time.

To match supplier import columns:

  1. Select your import file as above.
  2. On the Supplier Import Columns page, the column headings from your imported list will appear in brackets, such as (Supplier Name). Use the drop-down list under each heading to select the matching field in simPRO.
  3. If you wish to leave a column out of the import entirely, select Ignore from its drop-down list.
  4. Click Import Supplier.

To add additional supplier contacts, continue to select the Secondary Contact options for each additional contact.

Check supplier import success

After uploading an import file, the Import Supplier Summary page will display a breakdown of your import, including Suppliers Added / Updated / Ignored and Customers Created.

If the import was not successful, an error message will appear.

If the import was successful, click supplier imported successfully to download a CSV file of imported supplier for your records.

After a successful import, it is recommended you manually check imported supplier to ensure data has imported correctly. To do this, go to People > Suppliers, click an imported supplier, and check Profile > Details for errors. Manually amend if required.
