Currencies

Overview

If you sell or purchase goods and services in different currencies, you can set up multiple currencies in simPRO.

Set up currencies

You need to enable multiple currencies to use them in simPRO.

To enable currencies:

  1. Go to System > Setup > Defaults > Financial.
  2. Under Multiple Currencies, select On.
  3. Click Save.

In the System Setup tab, the Currencies option will now be available.

 Add a currency

To add a currency:

  1. Go to System > Setup > System Setup > Currencies.
  2. Select a currency from the Available Currencies drop-down list.
  3. Click Add.
  4. Click Save or Finish.
 Update the exchange rate

The exchange rate shown is the conversion value from the international currency to your default currency.

For example, if AUD is your default currency and GBP is added to the currency table, the exchange rate displayed would be 1 GBP to 1 AUD.

To update the exchange rate:

  1. Go to System > Setup > System Setup > Currencies.
  2. Click the blue refresh icon in the Exchange Rate column.
  3. Alternatively, enter the Exchange Rate manually,
  4. Click Save.
 Delete a currency

To delete a currency:

  1. Go to System > Setup > System Setup > Currencies.
  2. Click the red delete icon on the currency you wish to delete.
  3. Click OK.

If you delete a currency that has been assigned to a customer, supplier, or catalogue item, the value of any associated transactions will no longer display in that currency.

 Assign a currency to a customer, supplier, or catalogue item

To assign a currency to a customer or supplier:

  1. Access the desired customer / supplier.
  2. Go to Profile > Settings.
  3. Select a Currency from the drop-down list.
  4. Click Finish.

You can also enter prices in multiple currencies for catalogue items.

The Currency cannot be changed after a transaction with a customer / a supplier has been made.
