System Setup Overview

Overview

The System Setup tab is located under System > Setup > System Setup. The settings here affect many functions in simPRO.

Service

System Setup tab options available in Service are:

Labour

Set up labour rates and Service Fees.

Materials

Set up Material Defaults and Pricing Tiers and units of measurement.

Security Groups

Set up Security Groups for employees and contractors.

Activities

Set up non-billable Activities.

Accounts

Set up cost centres, Tax Codes and payment methods.

Status Codes

Set up Status Codes and Automatic Triggers.

Response Times

Set up Response Times to specify a job's urgency.

Custom Fields

Set up custom fields to record additional information.

Tags

Set up tags to label and filter customers and projects.

Customer Profile

Set up customer profiles.

Zones

Set up Zones for efficient scheduling.

Scripts

Set up scripts to save time with correspondence.

Plant Builder

Create Plant Types.

Archive Reasons

Set up Archive Reasons for quotes and jobs.

API

Read about simPRO's application programming interface for third-party software developers, or set up applications.
Enterprise

System Setup tab options available in Enterprise are:

Labour

Set up labour rates, service fees, labour fit times, Prepaid Labour Types, and Prepaid Blocks.

Materials

Set up pricing tiers, purchasing stages, units of measurement, and inventory tracking.

Security Groups

Set up Security Groups for employees and contractors.

Activities

Set up non-billable Activities.

Accounts

Set up cost centres, business groups, tax codes, payroll categories, payment methods, and accounting categories.

Status Codes

Set up Status Codes and Automatic Triggers.

Response Times

Set up Response Times to specify a job's urgency.

Custom Fields

Set up custom fields to record additional information.

Tags

Set up tags to label and filter customers and projects.

Notifications

Set up notifications to send emails or SMS when certain workflow actions are carried out or need attention.

Tasks

Set up task templates and categories to organise regular tasks.

Teams

Set up teams to quickly schedule technicians and plant items and avoid scheduling conflicts.

Customer Group

Set up customer groups.

Customer Profile

Set up customer profiles.

Zones

Set up Zones for efficient scheduling.

Scripts

Set up scripts to save time with correspondence.

Customer Assets

Set up asset types and service levels.

Plant Builder

Create Plant Types.

Archive Reasons

Set up Archive Reasons for leads, quotes and jobs.

Currencies

Set up currencies and refresh exchange rates.

API

Read about simPRO's application programming interface for third-party software developers, or set up applications.
