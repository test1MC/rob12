Import Sites

Overview

In simPRO, you can manually add Sites to the system through People > Create New Site, or save time and reduce data entry errors by importing a site list.

You may wish to test the import process with one or two sites before importing an entire list.

Prepare to import sites

To prepare to import sites into simPRO:

  1. Download and use simPRO's Site Import Template for:
    • Australia
    • New Zealand
    • United Kingdom
    • United States
  2. Download and use simPRO's Site Import Template.
  3. Download and use simPRO's Site Import Template.
  4. Download and use simPRO's Site Import Template.
  5. Download and use simPRO's Site Import Template.
  6. Alternatively, create your own import list. Leave out any fields that do not apply to your sites.
  7. Save your file in your system as a compatible file type: CSV (.csv like the Site Import Template above), text (.txt) or a zip file (.zip). Consult your spreadsheet software for file saving instructions if required.
  8. Separate your data using the same or similar headings as the Site Import Template.
  11. If your list contains customers that already exist in simPRO, make sure their names match exactly so the sites will automatically link with each customer in simPRO.

simPRO can recognise and import all the headings and information listed in the Site Import Template. Site names are compulsory.

If sites were created during an initial accounting package import, be careful not to import those sites again or duplicate site cards will be created.

If a site has a numeric-only custom field in simPRO but there is non-numeric data entered in the field in the site list, you will receive an error warning and the import will fail. Numbers must be formatted as specified in System > Setup > Defaults > System > Number Thousands Separator / Negative Number Format to import successfully.

Import a site list

After preparing your site list, work through the following steps to import your list to simPRO.

Adjust site import settings

To adjust site import settings, go to Utilities > Import > Sites.

The options available include:

First Record Is

Select Header Record if your list's first row contains column titles, or select Data Record if your list's first row contains specific site details.

If Site Already Exists

In case simPRO finds duplicated sites (sites with the same name), select whether you want to Do Nothing or to Update Site Details for the existing site in simPRO.

Create New Customer

Select Yes if you want to create a new customer card file for customers associated with a site. Select No if you do not wish to create new customers. Sites do not have to be associated with customers, but when creating a quote, job or lead, after choosing a customer, you will need to click Site > All to choose from a list of all sites.

If Custom List Value Not Found

If any of the entries are a drop-down list in simPRO (for instance, the State field is a drop-down list with QLD, NSW, SA etc), and your file contains any additional values (or 'custom list values') not available in simPRO's list, select Create New List Option to add the additional value to the drop-down list in your site card file. Otherwise, select Reject and Warn to ignore the field and receive a warning.

Select an import file

To select an import file:

  1. Go to Utilities > Import > Sites.
  2. Adjust site import settings, then click Browse / Choose File.
  3. Locate your site import file and click Open.
  4. Click Next.

Match site import columns

After selecting an import file, manually match or check your list columns with the fields available in simPRO. You may wish to have both simPRO and your list open at the same time.

To match site import columns:

  1. Select your import file as above.
  2. On the Site Import Columns page, the column headings from your imported list will appear in brackets, such as (Site Name). Use the drop-down list under each heading to select the matching field in simPRO.
  3. If you wish to leave a column out of the import entirely, select Ignore from its drop-down list.
  4. Click Import Sites.

To add additional site contacts, continue to select the Secondary Contact options for each additional contact.

Check site import success

After uploading an import file, the Import Site Summary page will display a breakdown of your import, including Sites Added / Updated / Ignored and Customers Created.

If the import was not successful, an error message will appear.

If the import was successful, click Sites imported successfully to download a CSV file of imported sites for your records. After a successful import, it is recommended you manually check imported sites to ensure data has imported correctly.

To do this, go to People > Sites, click an imported site, and check Profile > Details for errors. Manually amend if required.

Amend site import errors

If your site import file does not contain site names, you will receive an error message.

Click OK, add a site names column and data to your site import file and start the import process again.

If your file contains other errors, an error message will display on the Import Site Summary page, and the table will detail how many errors were found.

Click click here to download a CSV file of import errors. The Error Text column will explain why entries failed. Amend your site import list accordingly and start the import process again.
