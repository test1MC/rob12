Purchasing Stages

Overview

Use purchasing stages to categorise your materials. A purchasing stage labels an item, so that when you create a purchase order for a certain purchasing stage, only items with the applicable purchasing stage label applied will be available to order.

This will give you a more efficient ordering process when purchasing materials for your projects, as you only need to order the materials you need for one particular stage at a time.

You can only assign one purchasing stage to a catalogue item.

Create a purchasing stage

To create a purchasing stage:

  1. Go to System > System Setup > Materials > Purchasing Stages.
  2. Click Create Purchase Stage.
  3. Enter a Name.
  4. Click Create.

To delete a purchasing stage, click the red delete icon. To delete multiple purchasing stages, select the desired purchasing stages, then click Delete.

 Apply a purchasing stage to catalogue items

To apply a purchasing stage to a catalogue item:

  1. Go to Materials > Catalogue.
  2. Locate and click the desired item in the catalogue groups and subgroups.
  3. Select the applicable Purchasing Stage from the drop-down list in the Details tab.

To apply a purchasing stage to multiple catalogue items at once:

  1. Go to Materials > Catalogue.
  2. Locate the desired items in the catalogue groups and subgroups.
  3. Select the check boxes for the items you wish to update.
  4. Alternatively, select Select All at the bottom of the page to update all items in a group / subgroup.
  5. Click Set Purchase Stage in the footer options.
  6. Select the applicable Purchase Stage.
  7. Click Save.

For more advanced users, large numbers of items can be updated with a purchasing stage when importing catalogue items. Contact simPRO Support with assistance with this feature.

 Order with purchasing stages

To order with purchasing stages, create a purchase order from within a job.

To create a purchase order with a purchasing stage:

  1. Access the desired job, then go to Stock > Order.
  2. Select the applicable Purchasing Stage.
  3. Create the purchase order as normal.

Alternatively, to create a purchase order for jobs with multiple cost centres:

  1. Open the desired job, then go to Project Overview > Materials.
  2. Select the cost centres you are creating the order for.
  3. Select the purchasing stage from the Select Purchase Stage drop-down list.
  4. Create the purchase order as normal.
