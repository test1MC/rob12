Create a Purchase Order

Overview

Create a purchase order to order materials from a supplier. You can create a Catalogue Purchase Order to order items that exist in your simPRO catalogue, an Item Entry Order to order items that are not in your simPRO catalogue, a Description Order to order items without entering quantities, or a PO Number to raise an order that you can add items to later. Items can be ordered for a specific job or to update stock.

Create a purchase order from the Materials menu

You can choose from different purchase order options:

Create a new catalogue order

To create a new catalogue order:

  1. Go to Materials > Create New Catalogue Order.
  2. Under Assign To, select whether you are ordering to a Job or to Stock.
  3. If Job is selected, search and select the desired job.
  4. Select a Stage from the drop-down list.
  5. Enter Optional information as desired.
  6. Click the Catalogue tab.
  7. Find each catalogue item you wish to order, add the number of items required, and click Add Items.
  8. Click the Order tab.
  9. Click Show Cheapest Supplier or Show Cheapest Order if you have more than one supplier linked to an item.
  10. Click Finish.

You can then add Attachments and email the purchase order.

 Create a new item entry order

This type of order is often used when ordering item(s) that are not in your catalogue. If you wish to order it again in the future, it is recommended to add that item as a catalogue item.

To create a new item entry order:

  1. ​Go to Materials > Create New Item Entry Order.
  2. Under Assign To, select whether you are ordering to a Job or to Stock.
    • If Job is selected, search and select the desired job.
  3. Select a Supplier.
  4. Select a Stage from the drop-down list.
  5. Enter Optional information as desired.
  6. Enter the Current Order Items details.
  7. Click Finish.

You can then email the purchase order.

 Create a new description order

Be mindful that this order type does not have quantities. It is mainly used for a particular item you may wish your supplier to provide for a certain specification, such as manufacturing a frame.

To create a new description order:

  1. Go to Materials > Create New Description Order.
  2. Under Assign To, select whether you are ordering to a Job or to Stock.
    • If Job is selected, search and select the desired job.
  3. Select a Supplier.
  4. Select a Stage from the drop-down list.
  5. Enter Optional information as desired.
  6. Enter a Description and Amount.
  7. Click Finish.

You can then email the purchase order.

Users without the relevant security group settings will not be able to view, approve or email purchase orders.

 Create a new PO number

Create a PO number to generate a purchase order without adding any items. You can add items to the order later when required.

To create a PO number:

  1. Go to Materials > Create New PO Number.
  2. Select a Storage Device.
  3. Under Assign To, select whether you are ordering to a Job or to Stock.
  4. If Job is selected, search and select the desired job.
  5. Select a Supplier.
  6. Enter Optional information as desired.
  7. Enter Supplier Notes and Private Notes as applicable.
  8. Click Finish.

You can then email the purchase order.

To add items to the PO number, locate the order in Materials > Purchase Orders > Pending and click to edit it. Click the Catalogue tab and add catalogue items, or click Add New Item in the Details tab to create new items and add them.

 Create a purchase order from within a job

Create an order from within a job, so that the purchase order is already assigned to the job. This enables better management of your committed costs.

Create a purchase order from the project overview tab

If you have a job with multiple cost centres, you can create a purchase order from the Project Overview tab.

To create a purchase order from the Project Overview tab:

  1. Open the job and click the Project Overview​ tab.
    • If you have single cost centre jobs enabled, after opening the job click Cost Centres in the breadcrumb at the top of the page, then click the Project Overview tab.
  2. On the Materials page, click Create Purchase Order.
    • Alternatively, select predefined items to add to the purchase order and click Create Purchase Order.
    • The selected items will be displayed on the Order Details page.
  3. Under Assign To, select whether you are ordering to a Job or to Stock.
    • If Job is selected, search and select the desired job.
  4. Select a Stage from the drop-down list.
  5. Enter any Optional information.
  6. Click the Catalogue tab.
  7. Select or search for a group name.
  8. Add the number of items required and click Add Items.
  9. Click the Orders tab.
  10. Click the Show Cheapest Supplier tab or Show Cheapest Order tab if you have more than one supplier linked to an item.
  11. Click Finish.

You can then add attachments and email the purchase order.

You can select a Stage if items in the catalogue are listed under different project stages.

 Create a purchase order from a cost centre

To create a purchase order from a cost centre:

  1. Open the job and click the Cost Centre List tab.
  2. Select the cost centre.
  3. Click the Stock tab.
  4. Click the Order sub-tab.
  5. Enter details on the Order Materials page.
  6. Click Create Purchase Order.
  7. Click the Catalogue tab.
  8. Click or search for the relevant group name.
  9. Add the number of items required and click Add Items.
  10. Click the Orders tab.
  11. Click Show Cheapest Supplier tab or Show Cheapest Order tab If you have more than one supplier linked to an item.
  12. Click Finish.

You can then add attachments and email the purchase order.

You can select from different purchase types:

Blank

Order an item which is not defined in the job. You can add items from your catalogue, or change your purchase order type.

Stock Still Required

Raise a purchase order based on the items already defined in the job, through assigned materials or existing purchase orders.

Purchase Order Number

Raise a purchase order number only.
Email a purchase order

To email a purchase order / purchase order number to a supplier:

  1. Access the desired purchase order, then click Forms.
  2. Select to send a purchase order Form with or without prices.
    • Alternatively, select Email Only to send an email with no form.
  3. Click View to preview the form.
  4. Select the Primary Contact that you wish to address the email to, and click to add add additional Contacts as desired.
    • Alternatively, manually enter an email address in the To field.
  5. Adjust the default Subject and Body if desired. The default is set up in System > Setup > Forms Setup > Purchase Order > Email.
    • Alternatively, click Clear Email to enter a new Subject and Body.
  6. Click Send > OK > OK.

Go to System > Setup > Forms Setup > Purchase Order > Display Options > Form Titles to enter a new title to display on the printable form document. The form will still retain its simPRO default title when you select it from the Form drop-down list.

 Warnings

Fully invoiced cost centre

A warning will appear when allocating materials to an already fully invoiced cost centre. This feature is useful in the following scenarios:

To ensure you have selected the right cost centre. If it is fully invoiced and you are raising an order, you may be ordering against the incorrect job or cost centre.

To ensure you do not add additional costs to a job which has already been fully invoiced. Otherwise you will lose the ability to claim or invoice additional costs.

In some cases you may still need to add costs to a fully invoiced job to reflect the true cost of the job.

Outstanding purchase order

A warning will appear when invoicing a job with an outstanding purchase order.

 Tips

Change purchase order type

On the Order Details page, you can change the purchase order type from the PO Type drop-down list.

Locked purchase orders

Once a purchase order has been issued, it is locked and cannot be edited. To edit, open the order and click Unlock.

After unlocking, select Assign To stock or a job before adding additional items. This ensures the new items will be allocated correctly.

Single cost centre jobs

When Enable single cost centre service jobs / quotes is enabled in System > Setup > System Setup > Defaults, opening a single cost centre job will take you straight into the cost centre.

To order from the Project Overview tab, click Cost Centre near the top of the screen.
