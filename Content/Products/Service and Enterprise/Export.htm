Export

Overview

Export simPRO data into a Comma Separated Values (.csv) or Tab Delimited (.txt) file to manage and analyse data from different areas of your business.

You need to have the applicable security group permissions enabled to export.

Export simPRO data

To export from simPRO:

  1. Go to Utilities > Export.
  2. In the Exports menu to the left, click the area of data you wish to export.
  3. Under Export Type, select:
    • Comma Separated Values (CSV) to export your data into a CSV (.csv) spreadsheet.
    • Tab Delimited to export your data into a text (.txt) file.
  4. Adjust the export options as required.
  5. You can also export customers, suppliers, employees, and contractors according to information entered in any custom fields you have set up.
  6. You can also export customers, suppliers, employees, and contractors according to information entered in any custom fields you have set up.
  7. You can also export customers, suppliers and employees according to information entered in any custom fields you have set up.
  8. Click Export.

When the export is complete, a blue link to the export file will appear. To download the file, right-click the link, click Save Target As / Save Link As, then save the file in the desired location.

Customer contact details

Go to Export > Customers to export customer contact details and settings using the following options:

Customer

Enter to export customers with a matching name.

What to Export

Select Customers to export details from Profile > Details in the customer card file, as well as information in Profile > Settings > Customer Profile.

Select Contacts to export contacts added in Profile > Contacts in the customer card file.

Customer Type

Export Customers, Leads or All.

Customer Type

Export Customers, Leads or All.

Email Address Filter

Select Show only Customers / Contacts with Email Address to export customers / contacts with an email address entered in Profile > Details / Profile > Contacts.

Select Show only Customers with No Email Address to export customers / contacts with no email address entered.

Alternatively, select All.

Active / Archived Filter

Select Show Only Active Customers to export customers currently active in simPRO.

Select Show Active and Archived Customers to export both active and archived customers.

Select Show only Archived Customers to export only archived customers.

State

Export customers belonging to one or more specific states.

Customer Created Date Range

Export customers created within a specific date range.

Customer Tag

Export customers with specific customer tags applied in Profile > Tags. Alternatively, select No Customer Tag.

Customer Group

Export customers belonging to a specific customer group. Alternatively, select No Customer Group.

Customer Group

Export customers belonging to a specific customer group. Alternatively, select No Customer Group.

Customer Profile

Export customers belonging to a specific customer profile. Alternatively, select No Customer Profile.

Default Invoice Method

Export customers according to their default invoice method. Alternatively, select All.

Account Manager

Export customers assigned to a specific account manager. Alternatively, select No Account Manager.

Other Export Options

To adjust which fields are included in the customer export and how they are displayed, select:

Include 'CLI' Prefix on Customer ID to indicate the ID belongs to a customer (client);

Include Customer Banking Details in Export as defined in Profile > Settings > Banking in the customer card file;

Include Customer Tags in Export as defined in Profile > Tags in the customer card file;

Include Customer Custom Fields in Export as defined in Custom in the customer card file.
