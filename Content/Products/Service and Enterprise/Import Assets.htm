Import Assets

Overview

Create assets in simPRO by importing asset lists into sites according to their asset type. When an asset list has been successfully imported, you can view the newly created assets in their relevant sites in Utilities > Customer Assets.

You can create and maintain asset lists according to their asset type in spreadsheets saved as CSV (.csv), text (.txt) or ZIP (.zip) files. You can also separate these files by site and service start date if desired.

Asset types, including their custom fields and relevant contracts, as well as sites must first be created in simPRO prior to importing the relevant assets.

Prepare an asset import list

To prepare to import assets into simPRO:

  1. Download the simPRO Asset Import Template.
  2. Alternatively, create your own import list, leaving out any fields that do not apply to your customer assets.
  3. Divide your asset lists according to their asset type as defined in the system, as different asset types are set up with different custom fields.
    • You may also wish to divide asset lists by site and service start date, depending on how you adjust your import settings.
  4. Save your files as compatible spreadsheet file types: CSV (.csv), text (.txt) or ZIP (.zip). Consult your spreadsheet software documentation for file saving instructions if required.
  5. Separate your data using headings as specified in the field requirements.
Field requirements

When importing an asset list, you can include information defined by custom fields specific to the asset type. To apply custom field information to your asset list:

  1. Locate and open your asset list.
  2. In simPRO, go to System > Setup > System Setup > Customer Assets > Asset Builder.
  3. Click the relevant asset type as defined in the list.
  4. Click the Custom Fields tab.
  5. For each custom field, add a new column to the asset list, heading each column with the relevant Field Name.
    • The Field Name headers must be identical to how they are formatted in simPRO. It is recommended to copy and paste field names to ensure a successful import.
  6. Enter data in the columns as required.
    • Similar to the Field Name, custom field data in the form of list options must be identical to how it is formatted in simPRO. If entering new list options, be sure to select Create New List Option in your import settings.
  7. When you have finished entering custom field data for that asset type and list, save your file and repeat the process with your other asset lists.

In addition to custom fields, you can import the following fields of information when uploading asset lists to simPRO:

Site ID

The Site ID / Site Name are mandatory columns but do not both need to be present to import successfully. The Site ID is available in the site card in simPRO and must be a numerical value. If left blank on any line, the line will be ignored and the asset will not be created in simPRO.

Site Name

The Site ID / Site Name are mandatory columns but do not both need to be present to import successfully. The Site Name must be identical to how it is defined in simPRO. It is recommended to copy and paste the names from simPRO to reduce error and import successfully. If left blank on any line, the line will be ignored and the asset will not be created in simPRO.

Service Start Date

This will be the date of the most recent inspection and will be used to calculate future service dates in simPRO. Dates must be formatted as specified in System > Setup > Defaults > System > Date Format to import successfully.

Alternatively, you can set a service start date for every asset in your list using your import settings, or for individual assets in simPRO after importing.

Walk Order

The order in which a technician expects to service assets on a site, if applicable. This is useful when servicing a large number of customer assets per site.

Contract Number

A contract number links an asset to a predefined customer contract in simPRO to track profitability of a contract or to provide a customer with specific pricing. This is recommended for Maintenance Planner users.

Alternatively, you can enter contract numbers for assets in simPRO afer importing.

Contract Name

A contract name links an asset to a predefined customer contract in simPRO to track profitability of a contract or to provide a customer with specific pricing. This is recommended for Maintenance Planner users.

Alternatively, you can enter contract numbers for assets in simPRO afer importing.

If an asset has a numeric-only custom field in simPRO but there is non-numeric data entered in the field in the asset list, you will receive an error warning and the import will fail. Numbers must be formatted as specified in System > Setup > Defaults > System > Number Thousands Separator / Negative Number Format to import successfully.

Import an asset list

After preparing your asset list, work through the following steps to import your list to simPRO.

Adjust import settings

To adjust asset import settings, go to Utilities > Import > Sites.

The options available are:

Site Match

Select By File to import assets to sites defined in the asset list import file. This is recommended when uploading a single import file containing multiple sites, however the site names in the file must be identical to those in simPRO.

Select Manually Select if you are uploading an asset list specific to one site.

Service Start Date

Select By File to set service start dates for assets as defined in the asset list import file. This is recommended when uploading a single import file containing different service start dates.

Select Manually Select if you are uploading assets with the same service start date.

First Record Is

Select Header Record if your list's first row contains column titles, or select Data Record if your list's first row contains specific asset data.

If List Value Not Found

If any assets contain a custom field list value that is not available in the list in simPRO, select Create New List Option to add the value to the drop-down list in the custom field for that asset.

Select Reject and Warn to ignore the field and receive a warning.

Select Asset Type

Select the type of asset in the list you are importing.

You may wish to test the import process with one or two assets before importing an entire list.

 Select an import file

To select an import file:

  1. Adjust asset import settings, then click Choose File / Browse.
  2. Locate and open your customer import file.
  3. Click Next.
 Upload an asset list and match import columns

To upload an asset list and match import columns:

  1. Go to Utilities > Import > Assets and adjust import settings as desired.
  2. Click Browse / Choose File to locate and open the desired asset list.
  3. Click Next.
  4. Use the drop-down lists to match your import columns with the fields available in simPRO.
    • If you wish to leave a column out of the import entirely, select Ignore.
  5. Click Import Assets.
 Check import success

If the import is successful, you will be taken to the Import Asset Summary.

Click Assets imported successfully to download a CSV of imported assets for your records.

If your asset list contains errors, an error message will display on the Import Asset Summary page, and the table will indicate how many errors were found. Click click here to download a CSV file of import errors. The Error Text column will explain why entries failed. Amend your asset import list accordingly and restart the import process.
