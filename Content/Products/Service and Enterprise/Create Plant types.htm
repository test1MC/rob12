Create Plant Types

Overview

To create plant items, you first need to create plant types using the Plant Builder in System Setup.

Plant types define categories of plant items, such as vehicles or office equipment, so that you can manage them easily. Plant types can be found under Utilities > Plant & Equipment, where you can then create individual plant items.

To add plant items to your database, you will need the appropriate security group settings.

Create a plant type

Use plant types to arrange your plant items in groups to manage your plant and equipment efficiently.

To create a plant type:

  1. Go to System > Setup > System Setup > Plant Builder.
  2. Click Create Plant Type.
  3. Enter the Plant Type Name.
  4. Set the Service Frequency:
    • Select and enter Hours, or alternatively;
    • Select a Date and select how often the service is performed.
  5. Click Custom Fields to add custom fields. These help to identify one piece of plant from another, for example, the make and model. If you do not wish to create any custom fields, click Finish now.
  6. To add a custom field, click Create Custom Field.
  7. Enter the Field Name.
  8. Select the Field Type:
    • Select List to create a list of items to select from.
    • Select Text to create a text field to type in.
    • Select Date to create a date picker to select from.
  9. Click Create.
  10. Add more custom fields as desired.
  11. Click Finish.

 View plant types

To view the plant types you've created and any plant items associated with them, go to Utilities > Plant & Equipment. Click a plant type to see the list of plant items. You can also create plant items here.
