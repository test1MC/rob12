System Defaults

Overview

To change system defaults, go to System > Setup > Defaults > System. Changing these settings will reduce the data entry and simplify system processes. After changing system default settings, click Save to update your changes.

Enterprise Defaults

General

Mandatory Fields

Make certain card file fields mandatory to improve data entry.

Date Format

Change the date format system-wide. This impacts scheduling.

Time Format

Change the time format system-wide. This will impact scheduling.

Number Thousands Separator

Change the format of numbers over a thousand system-wide. This will impact imports and exports of data, for example, a CSV with spaces may result in numbers being treated as text in your spreadsheet.

Negative Number Format

Change the format of negative numbers system-wide. This will impact imports and exports of data.

System Alert Time

Set alerts to trigger a certain number of days prior to the due date.

No. Rows on Screen

Set the number of rows viewable on screen. This will impact the speed of page loading.

Licence Release

Specify the time limit on an inactive user before their licence is released to another user waiting to log in.

Service Jobs and Quotes

Select the check box to enable a single cost centre for service jobs and quotes in conjunction with the default cost centre. This speeds up the service job or quote creation workflow by skipping immediately to the cost centre after the customer and site details are entered.

Order Stage

Select the Purchase Order Stage that will be the default upon creation. Select Restrict Unapproved Orders to allow only approved orders to be viewed.

Prepaid Time

Select Customer & site based to share the credited Prepaid Time across all the customer sites or select Customer based to manage each site independently within that customer.

Jobs

Create Job

Set the default newly created job as Project or Service.

Default Description Field

Select whether the Overview or the Cost Centre is the default for the description upon job creation.

Service Job Cost Centre

Select the default cost centre for service jobs. Select Update all customers to add this default to all customers. This will also be the default for newly created customers.

Warranty Job Cost Centre

Select the default cost centre for warranty jobs. This is used in conjunction with Automatic Triggers to change warranty jobs to an alternate Job Status after completion. Useful when warranty billing or follow-up processes differ from standard jobs.

Invoice Job

Select Move to Invoiced Jobs to change the stage after fully invoicing a job. Select Leave in Current Stage to leave the job in the current stage.

Lock Job

When a job is locked, select if this locks the Price & Items or Price & Budgets.

Quotes

Create Quote

Set the default newly created quote as Project or Service.

Default Description Field

Select whether the Overview or the Cost Centre is the default for the description upon quote creation.

Quote Stage

Select the stage of a quote upon creation. Select Restrict Unapproved Quotes to allow only approved quotes to be viewed in the customer portal.

Move Schedules to Job on Converting

Select Yes to send scheduled hours on a quote to the job when the quote is converted. This allows you to track the costs of preparing the quote prior to winning the job.

Invoices

Invoice Stage

Select the default invoice stage upon invoice creation as Approved or Pending. Select Restrict Unapproved Invoices to allow only approved invoices to be viewed.

Customer Assets

Customer Assets Auto Assign to Jobs

Select On to automatically assign any assets present on the site to the job upon job creation.

Stock to Customer Asset Create Stage

Select Allocation to Job to alert the user to create assets when catalogue items marked as assets are allocated to the job. Select Receipting of Order to alert the user to create assets when items marked as assets are receipted.

Save Maintenance Planner Descriptions to

Select to add maintenance planner descriptions in the Overview, Cost Centre or None.

Customers

Default New Customer as

Select to the default for new customers to be Company or Individual.

Customer Type Conversion to Customer on

Select to convert leads to customers upon Quote Creation or Job Creation.
