Attachments

Overview

In simPRO, attachments can be added many areas, including quotes, jobs, items and people. You may wish to use attachments to easily store copies of agreements or contracts, images of items, and other associated documents.

Attachments are managed through the Attachments tab in a quote, job, customer card file etc. This tab does not appear until the project has been saved at least once.

Access to creating, editing and deleting attachments and attachment folders may be restricted by a user's security group settings.

Changes to attachments will update in Connect after syncing. Read more on using and renaming attachments in Connect.

Add an attachment

You can add an attachment by dragging and dropping a file into the Attachments tab, or by manually uploading it through the Browse button. A progress bar will appear while the file is uploading.

To drag and drop an attachment:

  1. Go the Attachments tab and open the desired folder as applicable.
  2. Drag a file from your computer and drop it in the Browse or drag attachments to upload box, or drop it on the desired folder.
  3. Click Save or Finish.

To upload an attachment by clicking Browse:

  1. Go to the Attachments tab and open the desired folder as applicable.
  2. Click Browse.
  3. Find and click the desired file.
  4. Click Open.
  5. Click Save or Finish.
 Delete an attachment

To delete an attachment:

  1. Go the Attachments tab and open the desired folder as applicable.
  2. On the item you wish to delete, click the red delete icon.
  3. Click OK.
  4. Click Save or Finish.
 Adjust attachment options

View attached images in a table

If you upload images of compatible file types as attachments on items and pre-builds, you can then set them to display in table view for easy visual identification.

To view attached images in a table:

  1. Go the Attachments tab and open the desired folder as applicable.
  2. Upload the desired image, or locate it in the list.
  3. Select Default image for that attachment.
  4. Click Save or Finish.
  5. Go to the applicable table, such as a group in the Catalogue.

If the image does not appear in the table, modify the table view:

  1. Click Modify Table View.
  2. Ensure Image is selected.
  3. Click Save or Finish.

Access to the Modify Table View feature may be restricted by a user's security group settings.

Make an attachment public

If you make an attachment public, it will be visible in the customer portal and in Connect.

To make an attachment public:

  1. Go the Attachments tab and open the desired folder as applicable.
  2. Upload the desired attachment, or locate it in the list.
  3. Select Public for that attachment.
  4. Click Save or Finish.

Email an attachment

If you select Email on an attachment, it will be available to email from the Forms tab. Selecting this option will automatically make the attachment public too.

To include an attachment when emailing a quote, invoice etc:

  1. Go the Attachments tab and open the desired folder as applicable.
  2. Upload the desired attachment, or locate it in the list.
  3. Select Email for that attachment.
  4. Click Save or Finish.

The attachment will now be available and automatically selected to email in the Select Attachments section of the Forms tab. To send an attachment on its own, select the Email Only form.

Attachments on items are for on-screen use only and will not carry over to quotes or jobs the item is assigned to.

 Rename an attachment

To rename an attachment after uploading it:

  1. Go to the Attachments tab and open the desired folder as applicable.
  2. On the file you wish to rename, click the pencil icon.
  3. Edit the Attachment Name as desired. See naming restrictions.
  4. Click Update.
  5. Click Save or Finish.

This change will reflect in Connect after syncing.

File name restrictions

Attachment file names cannot include special characters, such as / \ ? % * : | . " < > and blank spaces. If a special character is entered when renaming a file, the unsupported character will be removed from the file name.

 Search for an attachment

To search for an attachment:

  1. Go to the Attachments tab.
  2. Enter search terms in the search bar under Attachments.
  3. Press Enter or click Search.
 Organise attachments

Organise your attachments by sorting them into folders and sub-folders. Click Home at any time to go back to the main attachment folder.

In Connect users can only see attachments, not folder structure.

Create an attachment folder

To create an attachment folder:

  1. Go to the Attachments tab and open the desired existing folder as applicable.
  2. Click Create Folder.
  3. Enter a Folder Name.
  4. Click Create.
  5. Click Save or Finish.

Delete an attachment folder

Attachment folders can only be deleted when they do not contain any attachments. If you wish to delete a folder, you must first delete all the attachments within it.

To delete an empty attachment folder:

  1. Go to the Attachments tab and open the desired folder as applicable.
  2. On the folder you wish to delete, click the red delete icon.
  3. Click OK.
  4. Click Save or Finish.

Rename an attachment folder

To rename an attachment folder:

  1. Go to the Attachments tab and open the desired folder as applicable.
  2. On the folder you wish to rename, click the pencil icon.
  3. Click Update.
  4. Click Save or Finish.

Move an attachment into a folder

To move an attachment into a folder:

  1. Go to the Attachments tab and open the desired folder as applicable.
  2. Locate the required attachment, then click and drag it into the desired folder.
  3. Click Save or Finish.

You can also drag and drop folders into other folders.

Switch between list and grid view

Change the way attachments and folders are displayed to suit your needs. In grid view, images of compatible file types and PDF documents are displayed as thumbnails.

Under the Create Folder button, click Switch to List View or Switch to Grid View to switch between views.

 Download attachments

You can download attachments and attachment folders as a compressed file (a .zip file).

When you click ZIP, the .zip file will contain all the attachments and folders on screen and their contents, retaining your folder and sub-folder structure. If you wish to download all your attachment folders for a certain project, make sure you are in the top Home folder when you click ZIP. If you only want to download the contents of a single folder, open that folder before clicking ZIP.

To download attachments:

  1. Go to the Attachments tab and open the desired folder as applicable.
  2. In the top right of the Attachments section, click the download icon.
  3. Click ZIP.

You can then extract the folders and attachments as desired.

To download an individual attachment, click its name to open it, then save the file to your device.

 Advanced

Save a quote or job as an attachment

When you email a quote or job form, you have the option to attach the form PDF to the quote or job itself. This feature is useful for keeping track of earlier versions of a quote or job if it is edited later.

To save a quote or job form as an attachment:

  1. Open the Forms tab in your quote or job.
  2. Select a Form type and adjust other details as required.
  3. Click View to preview the PDF, then close and return to the Forms page.
  4. Click Attach.
  5. Click Save or Finish.

Attach a signature image file

If you attach an image of a signature to an employee or contractor's card file, you can later insert that image as a signature on quote and job detail forms.

Image file type restrictions

To view an image in a table, as a thumbnail in grid view, or in a preview window, it must be one of the following compatible file types:

  • .jpg or .jpeg
  • .png
  • .gif

You can also view TIFF images (.tif or .tiff) as thumbnails when in in grid view.
