Customers

Overview

In simPRO, a customer is an entity, such as a company or individual, that receives your invoice for the work performed. You can import customers through Utilities > Import > Customers, or you can create customers.

You can then apply specific response times, payment methods, preferred technicians and labour and material rates, as well as use customer profiles, groups, and tags to categorise and report on customers.

To enable customers to view their own quotes, jobs, invoices and tasks, pay invoices online, edit contact and site details and book quotes and jobs, adjust their customer settings to give them access to the Customer Portal.

View customers

To view customers, hover over People in the main menu, then click Customers in the Go To sub-menu.

To quickly locate a customer, click the starting letter of the customer name above the search bar or simply enter the name in the search bar and click Search.

If you have a large customer database, you may need to narrow your customer search results using the Advanced filter options. You can filter customers according to Customer Status, Customer Type, Account Manager, State, Customer Group, Customer Profile and Tags.

 Create a customer

To create a customer:

  1. Hover over People, then click Create New Customer.
  2. Alternatively, go to People > Customers > Create Customer.
  3. Select whether the customer is a Company or Individual.
  4. Enter a Company Name / First Name and Last Name.
  5. ​​If the name is unique in the system, a green tick will appear.
  6. Enter remaining Company / Individual Details as required.
  7. If the customer is only a prospective customer rather than one with active quotes or jobs, select Lead under Customer Type.
  8. If the customer is only a prospective customer rather than one with active quotes or jobs, select Lead under Customer Type.
  9. Enter Street Address details.
  10. If you will be sending correspondence, such as invoices, to a different address, enter a Postal / Billing Address. Otherwise, select Same as Street.
  11. Select Create a new site with this address if desired.
  12. AU: Enter the Suburb and select the State to automatically generate the Postcode.
  13. UK: Enter the Postcode, then click outside the text box to generate a selection of addresses.
Add contacts Complete custom fields Assign response times

In simPRO, a response time is the time taken between logging and scheduling a job. You can assign a response time to a customer, then select the desired response time when creating a job to indicate a job's urgency. You can also provide response time reports to your customers.

You need to create response times in your system setup prior to assigning them to a customer.

To assign response times to a customer:

  1. Create or access a customer as normal.
  2. Click Profile > Response Times.
  3. Select a desired response time from the drop-down list, then click Add.
  4. Adjust the Days, Hours and Minutes of the response time if desired.
  5. Click the tick icon to select a response time as a Default.

Select Remove Default?, then click Save to unmark all response times as Default.

To adjust customer settings, create or access a customer as normal, then go to Profile > Settings. Adjust Customer Profile and Banking settings as desired, then click Save or Finish.

Adjust Customer Profile settings using the following options:

A green tick will appear if the Username is unique in the system, as well as when you enter a strong Password. A yellow tick will appear if the password meets the minimum requirements. If the password is weak and does not meet the requirements, a red cross will appear.

Customer Group

Apply customer tags

Apply customer tags created in System > Setup > System Setup > Tags > Customer Tags to categorise and report on customers. You can also filter customers using tags in the Advanced customer search.

You can apply as many tags as you wish and use them to track customers with special requirements, how customers were recommended to your business, whether a customer subscribes to your newsletter etc.

To apply customer tags:

  1. Create or access a customer as normal.
  2. Go to Details > Tags.
  3. Click the Available Tags you wish to assign, or alternatively, click Add All.
  4. Click Assigned Tags to remove them.

Set customer-specific material and labour rates

Set default material and labour rates at the customer level in the Rates tab to assist in calculating costs when creating quotes and jobs.

You can also manage and create customer contracts and record prepaid time blocks that you have sold to the customer by going to Rates > Prepaid Time.

Adjust general defaults

Go to Rates > General in a customer card file to select a Service Fee to automatically apply to all new jobs created against the customer and override the default in System > Setup > System Setup > Labour > Service Fees.

You can also select a customer-specific Part Tax Code and Labour Tax Code to apply to quotes and jobs. Alternatively, select Use Part Tax Code / Use Labour Tax Code to apply the individual item tax code.

If applicable, enter a Discount or Fee percentage against the customer.

 Add labour rates

Go to Rates > Labour in a customer card file to add labour rates defined in System > Setup > System Setup > Labour > Labour Rates. You can also override the system default labour rate by selecting a customer-specific default.

To add labour rates against a customer:

  1. Create or access a customer as normal.
  2. Go to Rates > Labour.
  3. Select a labour rate from the drop-down list, then click Add.
  4. To specify a labour rate as the customer Default, click the tick icon.

Click the red delete icon to remove a labour rate.

 Adjust material defaults Enterprise

Go to Rates > Materials to select a customer-specific Pricing Tier as created in System > Setup > System Setup > Materials > Pricing Tiers.

You can also clear the Use Pricing Tier Default / Use Default check box to apply a customer-specific Default Markup. These settings can be overridden at the job / quote level.

 Service

Go to Rates > Materials to change the default markup if desired. Clear the check box and enter a percentage to use a markup different from the material default.

 View the customer history

Use the History tab in a customer card file to view a history of all transactions raised against the customer in simPRO.

 Associate sites with a customer

Use the Sites tab to create or add existing sites to associate with a customer. This is useful if you perform works at multiple job sites for the one customer.

To create a new site to associate to a customer:

  1. Access the desired customer, then go to the Sites tab.
  2. Click Create Site.
  3. Enter details as required, then click Finish.

To add an existing site in simPRO to a customer, access the customer card file, go to Sites, then search and select the Site Name and click Add.

You can also search for and view customer sites on a simTRAC-generated map by selecting the desired sites, then clicking Map Sites.

You can also associate customers to sites in the Customers tab within a site card file.

 Create customer notes

Use customer notes to maintain a running record of any correspondence logged against the customer, and assign employees to follow up correspondence if required. Categorise notes by Reference to their subject, such as a lead, quote, job, recurring job, invoice or campaign, or to the correspondence type.

You can import customer notes by going to Utilities > Import > Customers > Customer Notes, or alternatively, create notes in the customer card file. Emails sent from simPRO will automatically create a note.

To create a customer note:

  1. Access the desired customer, then click the Notes tab.
  2. Enter an appropriate Subject.
  3. Select a Reference to categorise the note and enter a note No. if desired.
  4. Enter the contents of the note.
  5. Select a Follow Up date and the employee you wish to Assign To following up the note.
  6. Click Save at the bottom of the note.
  7. Click Save or Finish in the top right.
 Manage customer attachments

 Advanced Create customer contracts

Use customer contracts in Enterprise to monitor profitability for particular services you provide your customers, for example, preventative maintenance.

When you create a contract in the customer card file, it essentially works as a secondary card file that can be selected as the Customer when creating a job, invoice, recurring job, or recurring invoice.

To create a new customer contract:

  1. Access the desired customer, then go to Rates > Contracts.
  2. Click Create Contract.
  3. Enter the Contract Name.
    • You may wish to use the customer name and a description of the contract, for example, 'ABC Lighting - Preventative Maintenance 2017'.
  4. Select a Start Date and End Date.
    • You will then be alerted when the contract is due to begin or expire.
  5. Enter a Contract No. and Value.
    • You will be alerted when you bill the customer for more than the entered Value.
  6. Enter an Email address.
    • This will be used for sending job / quote / invoice forms where the contract is applied if no other customer contact has been added.
  7. Enter Notes as desired.
  8. Click Save Contract.

Click a contract to edit it, or alternatively, click Options > View Contract. To remove a contract, click Options > Remove Contract.

To apply a specific Pricing Tier and Default Markup for materials to a contract, access the contract, then click the Materials sub-tab, make changes as desired, and click Save Contract.

To adjust a contract for inflation, access the contract, then click the Inflation sub-tab. Under Existing Inflation Rates, select the Start Date, enter the inflation percentage rate, then click Save Contract.

If you have purchased and enabled Maintenance Planner, you can also associate service level pre-builds to contracts to bill the customer for preventative maintenance.

 View and email the customer statement

To view the customer statement, go to People > Customers, then click Options > View > Statement on the desired customer. Alternatively, access the desired customer, then go to Profile > Details, and click Options > View Statement.

You can then view the statement Summary and Invoices created against the customer. You can also filter invoices by Start / End Date, view Current Invoices or All Invoices, apply payments to multiple invoices at once, and print / email multiple invoices, the customer statement, an overdue letter, or a final overdue letter.

To apply payments to multiple customer invoices at once, select the desired Invoices, click Pay Invoices, then apply the payment as normal.

Print / email multiple invoices

To print or email multiple invoices:

  1. Access the desired customer statement.
  2. Select the Invoices you wish to print / email.
  3. Alternatively, you can Select All.
  4. Select the desired Invoice Form.
  5. Click Print / Email Invoices.

You can then view which email addresses the invoices have been sent to, and if any invoices failed in sending.

Emailed invoices will automatically send to the primary contact for invoices as set in the customer card file.

Email the customer statement / overdue letter / final overdue letter

To email the customer statement, overdue letter, or final overdue letter:

  1. Access the desired customer statement.
  2. Click Forms.
  3. Select the desired Form.
  4. Click View to preview the statement / overdue letter / final overdue letter.
  5. Select a Primary Contact to address the form to and click to add additional Contacts if desired.
    • Alternatively, you can manually enter email addresses and separate them with a comma.
  6. Adjust the default Subject and Body if desired. The default is set up in System > Setup > Forms Setup > Statement.
    • Alternatively, click Clear Email to enter a new Subject and Body.
  7. Click Send > OK to confirm.

Go to System > Setup > Forms Setup > Statement / Overdue Letters > Display Options to enter a new form title to display on the printable form document. The form will still retain its simPRO default title when you select it from the Form drop-down list.
