Report Subscriptions

Overview

Use report subscriptions to automatically email regular reports.

Add a report subscription

To add a report subscription:

  1. Go to Settings > Report Subscriptions.
  2. Click New Subscription.
  3. Select the Report Type and Report Frequency.
  4. Under Vehicles, select which vehicles you would like the report to apply to.
  5. Under Recipients, select who you would like to send the report to.
  6. Different report types may have additional details you can select.
  7. Click Create.

 Edit a report subscription

To edit a report subscription, click the edit icon in the far right column of the report subscription you would like to edit. Click Update to save changes.

To remove a report subscription, click the delete icon next to the edit icon.

Deleting a user will also delete all associated report subscriptions.
