Invoice Stages

Overview

Many businesses require an invoice to be approved internally before it is sent to customers, to ensure quality and accuracy of billing. To assist with this process, invoices in simPRO can be set to have two approval stages: Pending and Approved. You can also restrict unapproved invoices so only approved invoices can be emailed to a customer.

Pending invoices have no impact on your accounting package, while approved invoices are sent through when exporting data to the accounting package.

Invoice stages can be adjusted for all invoice types: deposits, requests for claim, progress claims, and tax invoices.

Adjust security group settings to control which users can approve invoices. You can also set up invoice value approval limits so users can only approve invoices below specified values.

These stages are optional, and if your workflow does not require invoice approval you can adjust default settings accordingly.

Set up invoice approval security group settings

Use security groups to restrict users' ability to approve invoices. You can either:

  • Grant access to approve all invoices.
  • Allow approval of invoices below a specified value limit.
  • Restrict approval of invoices altogether.

To set up invoice approval security group settings:

  1. Go to System > Setup > System Setup > Security Groups > Actions > Invoices.
  2. Select the Approve Invoices check box as desired.
  3. If you select Approve Invoices, choose whether to allow approval for All invoices or whether to set a Limit. Enter a Limit as applicable.
  4. Click Save or Finish.

Next you need to set a default invoice stage.

Set a default invoice stage

Set whether every new invoice's stage starts as Pending or Approved.

To set a default invoice stage:

  1. Go to System > Setup > System Setup > Defaults > System.
  2. Select Approved or Pending from the Invoice Stage drop-down list.
  3. If applicable, select the Restrict Unapproved Invoices check box (see below for details).
  4. Click Save.

If your business requires an invoice approval process, set the default stage to Pending and select the Restrict Unapproved Invoices check box.

Then when an employee without approval permission creates an invoice, they will not be able to select the Approved invoice stage, and will not be able to view the Forms tab to email the invoice. A warning will also appear stating the invoice needs to be approved.

The Forms tab will only appear after the invoice stage is changed to Approved.

If your business does not require an invoice approval process, set the default stage to Approved.

Select an invoice stage

To select an invoice stage, if your security group settings allow:

  1. Invoice a job as usual.
  2. Select an Invoice Stage from the drop-down list.
  3. Make other changes as desired.
  4. Click Finish.

Approve and email a pending invoice

To approve and email a pending invoice (if your security group settings allow):

  1. Open the pending invoice.
  2. Click Unlock.
  3. Select Approved from the Invoice Stage drop-down list.
  4. Click Finish.
  5. Click the Forms tab and email the invoice as usual.
  6. Click Finish.

Reject a pending invoice and change details

If a pending invoice is rejected because details need to be adjusted, there are two options.

To change the amount claimed on a pending progress claim or request for claim:

  1. Open the pending progress claim or request for claim.
  2. Click Unlock.
  3. Adjust the Claim Amount as desired.
  4. If applicable, change the Invoice Stage to Approved.
  5. Click Finish.

If you want to change the total value on a progress claim or request for claim, you will need to delete the invoice and any other invoices associated with the cost centre for that job, and start again.

Before deleting an invoice, check that it is a pending invoice and not approved, as approved invoices have already been sent to your accounting package.

To change the total value on a pending invoice:

  1. Delete the pending invoice, and any other invoices associated with the cost centre for that job, by clicking Options > Delete Invoice > OK.
  2. Open the job, click Unlock and make changes as desired.
  3. Create a new invoice for the job.

Advanced

How invoice stages affect reporting

In the Sales Invoices report, you can filter by Invoice Stage.

In these reports, invoice dates and amounts billed only include information from approved invoices:

  • Aged Receivable report
  • Items Sold report
  • Contract Profitability report
  • Profit and Loss Summary report
  • Transactions Ready for Export report.
