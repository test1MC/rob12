Invalid GPS Data

Invalid GPS data is when simTRAC is not receiving a sufficient GPS signal from the tracker in a vehicle.

An orange exclamation icon next to your vehicle indicates a poor GPS signal. Possible causes include:

  • The vehicle is parked in garage or underground car park. If the vehicle has GSM coverage it will still give the ignition status for when the vehicle was turned on and off.
  • The GPS antenna has been installed in an incorrect place.
  • The tracker has been tampered with.

You will still receive updates from a tracker while the GPS location is invalid. Hover over the orange exclamation icon to view the number of updates.

The icon will disappear once GPS signal improves.
