Asset Custom Fields, Test Readings and Attachments

Overview

Use custom fields, test readings and attachments to store different types of information against an asset type for efficient asset management.

Custom fields

Use custom fields to store information about an asset according to its asset type, for example, serial numbers, asset IDs, makes, models, installation and warranty dates, or locations on site.

You can create five types of fields for asset types:

Text

Enter a small amount of text.

Date

Select a date from a date picker.

List

Choose from a drop-down list of options.

Barcode

Scan or enter a barcode.

Numeric

Enter numeric-only text. You can also enter decimal places and minus signs.

Numbers entered in Numeric ​custom fields will be automatically formatted according to settings in System > Setup > Defaults > System > Number Thousands Separator / Negative Number Format.

Create a custom field

To create a custom field for an asset type:

  1. Go to System > Setup > System Setup.
  2. In the left column, hover over Customer Assets, then click Asset Builder.
  3. Click the asset type you wish to create a custom field for, then click the Custom Fields sub-tab.
  4. Click Create Custom Field.
  5. Enter a Field Name.
  6. Select Mandatory / Required to make the custom field mandatory to complete.
  7. Select a Field Type.
    • For the List Field Type, click the Add icon to add more options to the list.
  8. Click Create.

When adding multiple fields to any given asset type, save regularly to prevent data loss.

 Test readings

When testing a customer asset out in the field, you can use test readings to record specific data in Connect, for example, electrical readings, pressure readings, temperatures etc. Test readings are then recorded as part of the asset history.

To enable technicians to record an asset's test results accurately using Connect, you need to create test readings in simPRO.

There are four types of fields you can apply to a test reading for technicians to complete:

Text

Enter a small amount of text.

Date

Select a date from a date picker.

List

Choose from a drop-down list of options.

Numeric

Enter numeric-only text. You can also enter decimal places and plus and minus signs.
Create a test reading

To create a test reading:

  1. Go to System > Setup > System Setup > Customer Assets > Asset Builder.
  2. Click the asset type you wish to apply a test reading to, then click the Test Readings sub-tab.
  3. Click Create Test Reading.
  4. Under Field Name, enter the name of the test reading, for example.
  5. Select Mandatory / Required if you wish for the test reading to be mandatory to complete when testing the asset in the field.
  6. Select a Field Type.
    • For the List Field Type, click the Add icon to add more options to the list.
  7. Click Create.
 Create a test reading when using Maintenance Planner

If you have purchased Maintenance Planner, you can assign test readings to specific service levels so that only necessary test readings appear for a particular service.

To create a test reading using Maintenance Planner features:

  1. Go to System > Setup > System Setup > Customer Assets > Asset Builder.
  2. Click the asset type you wish to apply a test reading to, then click the Test Readings sub-tab.
  3. Click Create Test Reading, then create a test reading as normal.
  4. Under Service Levels:
    • Select All to apply the test reading to all service levels.
    • Select Manual to apply the test reading to specific service levels assigned to the asset type.
      • Select service levels from the drop-down list to add. Click the red delete icon to delete.
  5. Click Create.

If you have not assigned service levels to an asset type, test readings that have been assigned to All will appear in the job card in Connect and Enterprise.

 ​Edit a test reading

To edit a test reading:

  1. Go to System > Setup > System Setup > Customer Assets > Asset Builder.
  2. Click the asset type that contains the desired test reading, then click the Test Readings sub-tab.
  3. Click the test reading and make changes as desired.
  4. Click Save.
