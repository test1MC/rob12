Import Supplier Invoices

Overview

Upload supplier invoices to add items to and fully receipt purchase orders. These items will then appear as billable items against the relevant unlocked jobs.

You can create and format these lists in CSV (.csv), text (.txt) or ZIP (.zip) files using a generic format or you can obtain files from your suppliers that have been specifically formatted for simPRO if your business is in Australia or New Zealand.

If the supplier invoice list contains items against a job that is locked in simPRO, the items will appear as non-billable against the job. It is recommended you manually review relevant jobs with pre-defined billable items after importing supplier invoices to ensure quantities and costs are consistent.

Suppliers must be created in simPRO prior to importing. You may wish to test the import process with one or two supplier invoices before importing an entire list.

Prepare to import supplier invoices

To prepare to import supplier invoices into simPRO:

  1. If you are importing a supplier invoice file in a generic format, download and add data to simPRO's Supplier Invoice Import Template.
  2. This template separates data with mandatory fields including Date, Order No., Invoice No., Part No., Description, Quantity Received, Unit Price Ex Tax / Line Price Ex Tax, and Unit Price Inc Tax / Line Price Inc Tax. You may also wish to include Packing Slip No. as a field.
  3. Save your file as a compatible file type: CSV (.csv), text (.txt) or a ZIP file (.zip). Consult your spreadsheet software documentation for file saving instructions if required.
  4. .

simPRO can recognise and import all headings and information listed in the Supplier Invoice Import Template.

Both generic and supplier-specific import files must contain purchase order numbers as defined in simPRO in the Order No. field.

If a supplier invoice has a numeric-only custom field in simPRO but there is non-numeric data entered in the field in the supplier invoice list, you will receive an error warning and the import will fail. Numbers must be formatted as specified in System > Setup > Defaults > System > Number Thousands Separator / Negative Number Format to import successfully.

Import supplier invoices in a generic format

After preparing your supplier invoice list, work through the following steps to import your list into simPRO.

Adjust supplier invoice import settings

To adjust supplier invoice import settings for a generic file, go to Utilities > Import > Supplier Invoices > Generic and search and select the supplier you wish to import invoices for in Select Supplier.

The available import options include:

First Record is:

Select Header Record if the first row of your list contains column titles, or select Data Record if the first row contains specific data.

If Part Not Found:

The import will compare the part numbers of items in the list file with those in the relevant purchase order. Select Create New Part to add and receipt new items not yet included in the relevant purchase order.

Select Reject and Warn to ignore additional items and receive warning in the Invoice Import Summary that the item was not added.

​Select an import file and match import columns

To import your supplier invoice list into simPRO:

  1. Go to Utilities > Import > Supplier Invoices > Generic and select the supplier you wish to import invoices for.
  2. Adjust import settings as desired.
  3. Click Browse / Choose File to locate and open the desired supplier invoice list.
  4. When the list file name appears, click Next.
  5. Use the drop-down lists to map your list file headings to match those in simPRO.
    • If you wish to leave a column out of the import entirely, select Ignore from its drop-down list.
  6. Click Next.

If the import is successful, you will be taken to the Import Invoice Summary.

To amend import file errors, refer to the Purchase Order and Problem list, make changes to the list file as desired, then repeat the import process.

To manually amend invoices that have imported successfully into simPRO, go to Invoices > Supplier Invoices > Receipts, click the relevant invoice and make changes as desired.

Import supplier-specific invoices

To import a supplier invoice list specifically formatted for simPRO in Australia or New Zealand:

  1. Go to Utilities > Import > Supplier Invoices and select the desired supplier from the drop-down list.
  2. Under Import Options - If Part Not Found:, select:
    • Create New Part to add and receipt new items in the file not yet included in the relevant purchase order.
    • Reject and Warn to ignore additional items and receive warning in the Invoice Import Summary that the item was not added.
  3. Click Browse / Choose File to locate and open the relevant supplier invoice list.
  4. When the list file name appears, click Next.

If the import is successful, you will be taken to the Import Invoice Summary. To amend any import file errors, refer to the Purchase Order and Problem list, make changes to the list file as desired, then repeat the import process.

To manually amend invoices that have imported into simPRO, go to Invoices > Supplier Invoices > Receipts, click the relevant invoice and make changes as desired.
