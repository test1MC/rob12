Scripts

Overview

Scripts are editable, pre-written responses for communicating with customers and sites, saving time and reducing errors.

Possible uses include:

Invoice Scripts

Statements qualifying trade or payment terms, or late payment fees.

Job Scripts

Job descriptions, qualifying statements, quality statements, technician notes, work requested, or work completed notes.

Quote Scripts / Lead & Quote Scripts

Overview description, scope of work, measure and quote statements, or qualifying statements.

Lead & Quote Scripts

Overview description, scope of work, measure and quote statements, or qualifying statements.

Quotes Scripts

Overview description, scope of work, measure and quote statements, or qualifying statements.

Orders

Acceptable delivery times or methods.

Tasks

Internal tasks or customer requirements.

Scope of work or qualifying statements.

SMS / Appointment / Vehicle

Directions, job dispatches, or work instructions.
Create a script

To create a script:

  1. Go to System > System Setup > Scripts.
  2. Click Create Script next to your desired script type.
  3. Enter a descriptive Name.
  4. In the Script box, enter the text you want your script to contain.
  5. For lead, quote and job scripts only, you can select a Cost Centre Default. The script will automatically populate on new transactions for that cost centre.
    • You can also assign cost centres below the Script box, so that this script will only be available for leads, quotes and jobs associated with those cost centres.
  6. For lead, quote and job scripts only, you can select a Cost Centre Default. The script will automatically populate on new transactions for that cost centre.
    • You can also assign cost centres below the Script box, so that this script will only be available for leads, quotes and jobs associated with those cost centres.
  7. For quote and job scripts only, you can select a Cost Centre Default. The script will automatically populate on new transactions for that cost centre.  
    • You can also assign cost centres below the Script box, so that this script will only be available for leads, quotes and jobs associated with those cost centres.
  8. Click Create.

Your new script will appear in the appropriate list.

 Edit a script

To edit an existing script:

  1. Go to System > System Setup > Scripts.
  2. Click on the script you wish to edit.
  3. Edit as needed, then click Save.
 Delete a script

To delete a script:

  1. Go to System > System Setup > Scripts.
  2. Find the script you wish to delete, then click the red delete icon on the right.
 Advanced options Adjust script order

You can adjust the order of your scripts to make the most frequently used scripts appear at the top of the list.

  1. Go to Setup > System Setup > Scripts.
  2. Use the drag handles drag and drop scripts into your desired order.
 Insert fields

When creating a script, click fields in the Insert Field menu to automatically add information such as a customer's name, today's date, or your company's address.

These fields will appear in square brackets, such as [Customer name], in your script. When you use the script to generate correspondence such as an invoice, the fields will automatically populate if the information is available.

 Hide scripts in simPRO Connect

If you have numerous scripts, you may want to save time by reducing the number of scripts field technicians have to scroll through. You can choose to show or hide certain scripts in Connect. By default, new scripts will not be available in Connect.

To show or hide a script in Connect:

  1. Go to Setup > System Setup > Scripts.
  2. Create a new script or click an existing script.
  3. Select or clear the Available in Connect check box as desired.
  4. Click Create or Save.

In the list of scripts, your selection will display in the Available in Connect column. Connect users will need to sync for the changes to apply.

 Make Overview the default cost centre

When creating or editing a script in simPRO Enterprise, you can select Overview as the Cost Centre Default. If selected, this script will be populated in the overview description field in leads, quotes and jobs.
