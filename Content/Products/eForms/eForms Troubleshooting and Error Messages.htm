Error messages

Error Message

Solution

simPRO Software status messages

Job Number does not exist or invalid format

Login to simPRO Software and verify job number. If correct, contact Help Desk.
Quote Number does not exist or invalid format

Login to simPRO Software and verify quote number. If correct, contact Help Desk.
Job locked

Return to simPRO Software and close the job.
Failed to set job lock.

Login to simPRO Software and lock the job.
Processing of JSON failed

The eForm JSON could not be retrieved or has an invalid format. Please contact the Help Desk.
Insert into database failed.

eForm will not be processed. Please contact the Help Desk.
No eForm data present.

eForms push to simPRO Software failed due to invalid JSON format or incorrect/empty parameters. Please contact the Help Desk.
Could not determine if simPRO import successful

Import may have been successful but eForms connection to simPRO Software may have dropped out. Check your connection and contact the Help Desk for further assistance.
Import error: Failed to attach PDF (enterprise issue)

Please contact the Help Desk.
Import error: Failed to prepare PDF (can't retrieve or access eforms pdf file)

Please contact the Help Desk.
Import error: Failed to attach image (enterprise issue)

Please contact the Help Desk.
Import error: Failed to prepare image (can't retrieve or access eforms image file)

Please contact the Help Desk.

eForms Status Message

Server returned non-valid JSON.

simPRO Software sent something that eForms doesn't understand or expect. Please contact the Help Desk.
Could not connect to host.

Can't connect to simPRO Software build. Verify your simPRO Software settings. If connected, please contact the Help Desk.
Form failed to unzip

The form that you are downloading has failed to unzip due to another user editing the form. Contact your office and request a notification when completed and try again. If the error continues please contact the Help Desk.
