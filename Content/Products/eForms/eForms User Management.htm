eForms User Management

Overview

Manage eForms users, security groups and extensions with simPRO Software in your system setup.

Manage security groups

You can manage the following permissions for the Employee and Supervisor security groups in eForms:

Portal Access

Log in to the eForms portal and view submitted forms. For those only using the eForms field app, this option can be cleared.

View All Submissions

View all submitted forms. If this option is cleared, the user will only see their own submitted forms.

Form Editor

Access the form designer to edit accessible forms.

View Users

View all eForms users.

Edit Users

Create and edit eForms users.

Form Designer

Access the form designer to create and edit form templates.

Edit Security Groups & Extensions

Edit security group permissions and enable or revoke simPRO integration This permission is usually only enabled for the Manager security group.

To manage eForms security group permissions:

  1. Go to System > Setup > Extensions.
  2. For each security group, select the check boxes for the permissions you wish to enable.
    • Clear the check boxes for the permissions you do not wish to enable.
  3. Click Update Security Groups.

The Manager security group must have full access to all eForms functionalities and therefore cannot be modified.

 Manage users

To view all eForms users, go to System > Setup > Users. You can then filter users by Name, Phone, Email or Security Group, or view deleted eForms users by selecting Show Archived Users Only.

Click Options to edit or archive a user.

You can edit a user's name, username and password, licensing, access to the Walk Me Through help widget and contact details. When you have finished making changes, click Submit User Changes.

Archiving a user will not remove their submitted forms from the eForms Portal.

Create an eForms user

To create an eForms user:

  1. Go to System > Setup > Users.
  2. Click Create New User.
  3. Enter a First Name and Last Name.
  4. Enter an Identifier if desired.
    • You may choose to enter their simPRO Employee ID to use for simPRO integration, or simply as a personal reference field.
  5. Enter the Username and Password to enable the user to log in to eForms depending on their security group permissions and licensing.
  6. If the user will be accessing the eForms app using an individual user licence, select Application Licence.
    • ​​​​Administrators of the eForms portal do not require a licence.
  7. Select the applicable Security Group.
  8. Enter a Phone number and Email.
  9. Click Create.
 Integrate eForms with simPRO

Connect eForms to simPRO Enterprise or simPRO Service to integrate your forms with simPRO data. You can then import forms to attach to jobs and quotes, as well as have simPRO data auto-complete within forms.

simPRO uses APIs that collect JavaScript Object Notation (JSON) data to integrate with eForms. To allow eForms to connect with simPRO, you need to authorise eForms to access your simPRO account.

To integrate eForms with simPRO:

  1. Go to System > Setup > Extensions.
  2. Click Connect to simPRO.
  3. If you are not currently logged in to your simPRO account, enter your Username and Password, then click Login.
  4. Click Authorise.

eForms will then be integrated with simPRO.

To cancel the connection between eForms and simPRO, go to System > Setup > Extensions and click Revoke.

 View the user log

Go to System > User Log to access the simPRO Documentation System Log.

Here you can access a record of all employee activity within eForms, including forms created, updated, submitted and archived, logging in and out, connecting with simPRO and adjusting eForms settings.

Use the Date Range and User Name filters to narrow results.
