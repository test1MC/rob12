Jobs Cost to Complete Report

Overview

Use the Cost to Complete report to see the projected cost to complete each project job. The report references both original and revised estimate budgets, actuals to date as well as additional forecasting to assist in managing any variances from those budgets. This report only shows project jobs that are pending or in progress and includes all activities within the job up to and including the date chosen.

View Cost to Complete report

To view the report:

  1. Go to Reports > View Reports.
  2. Hover over Jobs and click Cost to Complete.
  3. Select the Date (Past & Including).
  4. Select whether to include Variations.
  5. Select whether to Show Revised Budgets.
  6. Select whether to include Overheads in Actuals.
  7. Select whether to include Committed Costs.
  8. Under View, select whether to view Operations or Financials.
  9. Filter the report by selecting from the drop-down lists.
  10. Click Search.
 Download the report as a CSV file

You can download the information in the report as a spreadsheet in CSV format, viewable in Microsoft Excel and other spreadsheet software.

To download the report:

  1. Generate the report as desired.
  2. Click Download CSV at the bottom of the page.
  3. View, print or save the CSV as required.
