Manage Groups

Overview

Use groups to organise vehicles. Vehicles can belong to more than one group. You can also restrict access to groups so that users can only view certain vehicles.

Add a new group

To add a new group:

  1. Go to Settings > Manage Groups.
  2. Click Add New Group.
  3. Enter the Group Name.
  4. Enter a Description, if desired.
  5. Under Vehicles, click the add icon to assign a vehicle. You can use the search bar to find a specific vehicle, or you can click Add All to assign all vehicles.
  6. Under Users, click the add icon to give a user access to this group. You can use the search bar to find a specific user, or you can click Add All to give all users access.
  7. Click Create.

To change group details, click the edit icon.

To remove a group, click the delete icon.
