Add New Sites

Overview

Create sites to mark areas of interest, such as job sites, project sites, employees' homes, suppliers, the workshop, the office etc.

You can also search and report on sites more easily by grouping them according to types. Sites appear as blue circles on the map and can be imported directly from sites saved in simPRO, as well as created manually.

Once you have created sites, you can set up alerts to receive an email or onscreen alert when a vehicle enters or leaves a site boundary.

Create a new site

To create a new site:

  1. Go to Map > GeoFences and click New Site.
  2. Enter the address of the site you wish to add in the search bar.
  3. Select the address from the drop-down list.
  4. Name the site.
  5. Select the site Type from the drop-down list.
  6. Select the Radius of the site from the drop-down list to determine the size of the site.
  7. Set a speed limit if you wish to create an overspeed alert for this site.
  8. Click Create.

The site will then appear on the map.

 Manage sites

Click Manage to Manage Geofences and Sites.

 Import sites from simPRO

Unlike geofences, existing sites can be imported directly from simPRO after simPRO Integration.
