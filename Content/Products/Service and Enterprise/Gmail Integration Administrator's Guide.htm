Gmail Integration: Administrator's Guide

Install simPRO Gmail Integration

This process can only be done by domain administrators.

To install the simPRO Gmail Integration app in Google:

  1. Log in to your Google account.
  2. Go to the simPRO Gmail Integration page in the Google Apps Marketplace by clicking here.

  3. Click INSTALL APP.

  4. Under Turn ON for, select the users or organisations to use this feature.
  5. Select the I agree to... check box if you agree to Google's Terms of Service.
  6. Click Accept.

  7. After the simPRO Gmail In... has been installed! screen appears, close that browser tab or window.

Return to the main Gmail Integration article.

 Edit simPRO Gmail Integration settings

This process can only be done by domain administrators.

To edit simPRO Calendar Integration app settings:

  1. Log in to your Google account.
  2. Click the Google Apps icon in the top right of the screen (the nine dots shaped into a square).
  3. Click the Admin icon.
  4. Go to Apps > Marketplace Apps > simPRO Gmail Integration.
  5. Click Settings and edit as desired.
 Uninstall simPRO Gmail Integration

This process can only be done by domain administrators.

To uninstall the simPRO Calendar Integration app:

  1. Log in to your Google account.
  2. Click the Google Apps icon in the top right of the screen (the nine dots shaped into a square).
  3. Click the Admin icon.
  4. Go to Apps > Marketplace Apps > simPRO Gmail Integration.
  5. Click the Remove App icon (the rubbish bin) in the top right.
  6. Click REMOVE APP.
