Response Times

Overview

In simPRO, a response time is the time taken between logging and scheduling a job. You can assign a response time to indicate a job's urgency.

For example, a customer might call with an urgent job. If you mark it as 'urgent' with a response time of two hours, it needs to be scheduled within two hours of the phone call, even if it is scheduled for the next day. An alert will appear on the simPRO dashboard to remind you of the job's response time.

To use response times, they must first be assigned to a customer, then selected when creating a job. If desired, you can also provide response time reports to your customers.

Create a response time

To create a response time:

  1. Go to System > Setup > System Setup > Response Times.
  2. Click Create Response Time.
  3. Enter a Name.
  4. Enter the desired Days, Hours or Minutes.
  5. Click Create.
 Edit a response time

To edit a response time:

  1. Go to System > Setup > System Setup > Response Times.
  2. Click the response time you wish to edit.
  3. Make changes as desired.
  4. Click Save.
 Delete a response time

To delete a response time:

  1. Go to System > Setup > System Setup > Response Times.
  2. On the response time you wish to delete, click the red delete icon.
  3. Click OK.
 Assign a response time to a customer

To assign a response time to a customer:

  1. Go to People > Customers.
  2. Click the customer you wish to assign to.
  3. Under the Profile tab, click Response Times.
  4. Select a response time from the drop-down list, then click Add.
  5. Click Finish.

Use the red delete icon to remove a response time from a customer.

Once you have assigned a response time to a customer, you can customise it for that customer as needed.

 Select a response time for a quote or job

After assigning response times to a customer, you can select a response time for their quotes and jobs.

To select a response time on a quote or job:

  1. Create a job or edit an existing job.
  2. On the Main job setup tab, select a Response Time from the drop-down list.
  3. Adjust the rest of the job as needed, then click Finish, or Next to continue adding job details.
 Advanced

Create a report on response times

If you want to show a customer your response times to their quotes and jobs, as you might be required by a contract, you can create a report.

To create a report on response times:

  1. Go to Reports > Jobs > Response Times.
