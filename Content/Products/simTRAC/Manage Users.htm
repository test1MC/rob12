Manage Users

Overview

Manage your users and what pages they can access. You can also send users a link to reset their password.

Add a new user

To add a new user:

  1. Go to Settings > Manage Users.
  2. Click Add New User.
  3. Enter the Username and Email Address.
  4. Enter the user's First Name and Last Name.
  5. Select the check boxes to allow the user to access different parts of simTRAC:
    • Maps and Reports: all users by default have access to maps and reports.
    • Company Settings: allow the user to access Company Details, Users, Groups, Alert Triggers, Geofences and Sites, Drivers, and Tags on the Settings page.
    • Maintenance Planner: allow the user to access Maintenance Planner.
  6. Select a country and timezone.
  7. Click Create.

The user will be sent an email with their username and a link that will allow them to set their password.

To change details for an existing user, click the edit icon.

To remove a user, click the delete icon.

 Send reset link

To allow an existing user to reset their password, click Send Reset Link next to their name.

 Groups

New users will automatically have access to the default vehicles group. To allow users to access specific groups, go to the Manage Groups page.
