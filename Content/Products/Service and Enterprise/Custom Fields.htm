Custom Fields

Overview

Custom fields can be used to store additional information such as a customer's alternate email, or alternate contact person. After they are created, custom fields can be found under the Custom tab in each record.

You can create the following types of fields:

Text

Enter a small amount of text.

Date

Select a date from a date picker.

List

Choose from a drop-down list of options.

Numeric

Enter numeric-only text. You can also enter decimal places and minus signs.

Numbers entered in Numeric ​custom fields will be automatically formatted according to settings in System > Setup > Defaults > System > Number Thousands Separator / Negative Number Format.

Create a custom field

To create custom fields:

  1. Go to System > Setup > System Setup > Custom Fields.
  2. Click the record you want the custom field to apply to (such as Customers).
  3. Click Create Custom Field.
  4. Enter a Field Name.
  5. Select the Field Type.
  6. Select Mandatory / Required to make it a compulsory field.
  7. For the List Field Type, click the add button to add more options to the list. Click the red delete icon to remove options from the list.
  8. Click Create.

An additional Show For check box will appear when creating custom fields for quotes, jobs, recurring jobs / invoices, and leads.

 Add custom fields to table view

You can add custom fields to the table view so you do not have to open each record. Modifying the table view will change the table view for all users, not just the logged-in user.

To add a custom field to the table view:

  1. When viewing a table, click Modify Table View next to the search bar.
  2. Select the check boxes for the desired custom fields.
  3. Click Finish.

You can select and view custom fields for the related section only. For example, a custom field created under Customers will only be available when creating or editing customers.

After you have added custom fields to the table view, you can sort the table by these fields by clicking the heading of the custom field column.

Tips

  • Use drag handles in Modify Table View to reorder fields and options.
  • You can have up to 15 columns displayed in table view.

Modify Table View can be limited to certain security groups. Only users with access to it can add or make changes to the table view.
