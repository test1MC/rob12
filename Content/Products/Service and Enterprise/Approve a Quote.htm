Approve a Quote

Overview

To approve a quote, access the quote as normal, then go to Details > Settings > Stage, then select Approved. Project quotes and service quotes with multiple cost centres can only be approved at the overview level.

Set up a quote approval workflow

Quote approval workflows vary between different businesses. If you have a specific quote approval workflow that involves different roles in your business, you can control who can approve quotes using security groups, or alternatively, allow employees to only approve quotes up to a set value.

To restrict quote approval permissions using security groups:

  1. Go to System > Setup > System Setup > Security Groups.
  2. Click the applicable security group.
  3. Go to Permissions > Actions > Quotes.
  4. To restrict employees assigned to the security group from approving any quotes, clear the Approve Quotes check box.
  5. Alternatively, to restrict the security group to only approving quotes up to a certain value, select Approve Quotes, then select Limit, and enter the value as desired.
  6. Click Save.

To adjust your quote approval workflow at the system-wide level:

  1. Go to System > Setup > Defaults > System > Quotes.
  2. Under Quote Stage, select:
    • In-Progress to locate created quotes in Quotes > Progress Quotes. This is useful if the majority of your quotes require considerable detail and time to be compiled.
    • Complete to locate created quotes in Quotes > Complete Quotes. This is useful for quoting workflows where one employee creates and builds the quote, and another employee, such as a sales manager, needs to review and approve it before sending it to the customer.
    • Approved to locate created quotes in Quotes > Approved Quotes. This is useful for quoting workflows where the quote is created, built, finalised and sent to the customer by the one user from start to finish.
  3. If desired, select Restrict Unapproved Quotes to restrict the Forms tab from appearing in Progress / Complete Quotes to prevent sending or printing unapproved quote forms.​
  4. Adjust other quote defaults as desired.
  5. Click Save.
