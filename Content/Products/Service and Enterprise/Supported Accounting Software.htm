Supported Accounting Software

Overview

Integrate simPRO with your accounting package to keep your financial data accurate and transparent.

Supported online accounting software packages

Australia and New Zealand

The following online accounting software packages are supported in simPRO in Australia and New Zealand:

  • QuickBooks Online
  • Xero
  • MYOB AccountRight Live

You need to check in your company accounting file to the MYOB cloud for MYOB AccountRight Live to be supported. The link will not be supported if the company file is stored on your local machine or server (checked out).

Supported desktop accounting software packages

Australia

The following desktop accounting software packages are supported in simPRO Enterprise in Australia:

  • MYOB AccountRight (V19)
    • If you have recently upgraded to MYOB V19.13, you may receive an error when trying to integrate with simPRO. To resolve this, go to the Setup tab in the accounting link and make sure that the MYOB Program File is linked to the new file.
  • Reckon Accounts
  • MYOB EXO (see our Partner Program for further details)
