Set up New Alerts

Overview

When a vehicle's tracker records an event, for example, the vehicle exits a geofence, you will be notified by an alert. There are three types of alerts you can receive when triggered by an event:

Email Alert

An alert will be sent to you immediately via email.

Onscreen Alert

An alert will appear as a pop-up box on your screen with an option to acknowledge or ignore the alert.

Onscreen Alert with sound

An alert will appear as a pop-up box on your screen accompanied by a siren sound with an option to acknowledge or ignore the alert.

 

You can only set up alert triggers for the alerts your trackers have been configured to send. To configure more alerts, contact our Support Team.

Set up alerts for events

To set up a new alert:

  1. Go to Settings > Manage Alert Triggers.
  2. Click Add New Trigger.
  3. Under Criteria, select the Event(s) you wish to receive an alert for.
  4. Select Vehicles or Groups to add the alert trigger to specific vehicles or groups, or simply select All Vehicles.
  5. Click the add icon to the right of the available vehicles / groups you wish to monitor using the alert trigger, or click Add All.
    • Alternatively, you can narrow your search results for vehicles / groups by entering information in the Filter... search bar.
  6. To enable the alert to trigger:
    • At all times, select All days and times.
    • During a specific schedule, select As per schedule and select days and times in the grid as desired.
  7. Select On-screen, On-screen (include sound) and / or E-mail to choose how you wish to be notified of the alert.
  8. If you select E-mail, click the plus icon on the available email addresses you wish to receive an alert.
  9. Click Create.

You can only select email addresses of simTRAC users when setting up alert notifications.

 Set up alerts for geofences and sites

You can create alerts for when vehicles enter or exit a geofence or site. You can also set up alerts for vehicles speeding within a specific geofence or site.

To set up an alert for a geofence or site:

  1. Create an alert as normal.
  2. Select Entry and / or Exit as desired.
  3. Under Geofences / Sites Available, click Add All to apply the alert to all geofences and sites.
    • Alternatively, search for specific geofences or sites and click the add icon to add them.
    • Hover over the pin icon to see what the geofence / site looks like on the map.
  4. Select options for Vehicles, Schedule and Notifications as normal.
  5. Click Create.

You can add or remove geofences and sites at any time by editing the alert.

Set up overspeed alerts for geofences and sites

To set up an overspeed alert for a geofence or site, you need to set the speed limit while creating the geofence or site. You can do this for existing geofences / sites by editing them.

To set up an overspeed geofence / site alert:

  1. Create an alert as normal.
  2. Select Overspeed.
  3. Under Geofences / Sites Available, click Add All to apply the alert to all geofences and sites.
  4. Alternatively, search for specific geofences or sites and click the add icon to add them.
    • Hover over the pin icon to see what the geofence / site looks like on the map.
    • Select options for Vehicles, Schedule and Notifications as normal.
  5. Click Create.
 Edit an alert

To edit an existing alert:

  1. Go to Settings > Manage Alert Triggers.
  2. Click the edit icon next to the alert.
  3. Make changes as desired.
  4. Click Update.

To remove an alert, click the delete icon.
