 Prepare the list for import

Once your import list has been created, you can prepare your list for import with the following steps:

  1. Save your file in your system as a compatible file type: CSV (.csv), text (.txt) or a ZIP file (.zip). Consult your spreadsheet software for file saving instructions if required.
  2. Separate your data using the same or similar headings as the import template.

If an employee / contractor has a numeric-only custom field in simPRO but there is non-numeric data entered in the field in the employee / contractor list, you will receive an error warning and the import will fail. Numbers must be formatted as specified in System > Setup > Defaults > System > Number Thousands Separator / Negative Number Format to import successfully.

Import an employee / contractor list

After preparing your employee / contractor list, work through the following steps to import your list into simPRO.

Adjust import settings

To adjust employee / contractor import settings, go to Utilities > Import > Employees or Employees / Contractors.

The import options include:

First Record Is

Select Header Record if your list's first row contains column titles, or select Data Record if your list's first row contains specific data.

If Employee Already Exists

In case simPRO finds duplicated employees / contractors (employees / contractors with the same name), select whether you want to Do Nothing or Update Employees Details for the existing employee / contractor in simPRO.

If Custom List Value Not Found

If any columns in the import file represent drop-down lists in simPRO (for example any custom fields), and your file contains any additional values (or 'custom list values') not available in the relevant list, select Create New List Option to add the additional value to the applicable drop-down list. Select Reject and Warn to ignore the field and receive a warning that the field was not created.

Employee Status

Select whether the upload file is an Employee or Contractor list.

Select an import file

To select an import file:

  1. Go to Utilities > Import >Employees, Employees / Contractors.
  2. Adjust import settings, then click Choose File / Browse.
  3. Locate your employee / contractor import file and click Open.
  4. Click Next.
 Match import columns

After selecting an employee / contractor import file, manually match or review your list columns with the fields available in simPRO. You may wish to have both simPRO and your list open at the same time.

To match import columns:

  1. Select your import file as above.
  2. On the Employee Import Columns page, use the drop-down lists under each heading to select the matching field in simPRO.
  3. If you wish to leave a column out of the import entirely, select Ignore.
  4. Click Import Employees.
 Check import success

After uploading an import file, the Import Employee Summary page will display a breakdown of your import, including Employees Added / Updated / Ignored.

If the import was successful, click Employees imported successfully to download a CSV file of imported employees / contractors for your records, with each line containing a simPRO-assigned Employee / Contractor ID.

After a successful import, it is recommended you manually check employee / contractor card files to ensure data has imported correctly. To do this, go to People > Employees / Contractors, click an imported employee / contractor, and check Profile > Details for errors. Manually amend if required.
