Field requirements

When you create a job import list or add data to the Job Import Template, you need to alter your data to suit the following field requirements:

  • The Customer ID / Customer Name are mandatory columns but do not both need to be present to import successfully. If these fields are left blank on any line, the line will be ignored and the job will not be created in simPRO.
    • The Customer Name must be identical to how it is defined in simPRO. It is recommended to copy and paste the names from simPRO to reduce error and import successfully.
    • The Customer ID is available in the customer card file in simPRO and must be a numerical value.
  • The Site ID / Site Name are mandatory columns but do not both need to be present to import successfully. If these fields are left blank on any line, the line will be ignored and the job will not be created in simPRO.
    • The Site Name must be identical to how it is defined in simPRO. It is recommended to copy and paste the names from simPRO to reduce error and import successfully.
    • The Site ID is available in the customer / site card file in simPRO and must be a numerical value.
  • Multiple Project Tags can be specified by separating each one with a comma. Therefore the specific tags cannot contain commas in their names or they will import incorrectly.
  • Customer Contact and Supplier Contact names must be in '{First Name} {Last Name}' format, for example, 'John Smith' instead of 'Smith, John'.
  • If jobs in the list have a specific Due Date, the format must be as defined in System > Setup > Defaults > System > Date Format.
  • The Job Type field for each line must be specified as Service, Project or Prepaid as applicable. If not specified, the job will default to the Service job type.

 Import a jobs list

To import a jobs list into simPRO:

  1. Go to Utilities > Import > Jobs.
  2. Click Browse / Choose File to locate and open your jobs list file.
  3. When the file name appears, click Next.
  4. On the Job Import Column Mapping page, use the drop-down lists to map your column headings against those available in simPRO.
  5. Click Finish.

If the import is successful, you will be taken to the Job Import Summary where you can view jobs created and any import errors.

To amend import errors, you may need to access the relevant jobs in simPRO individually and manually make changes as desired.
