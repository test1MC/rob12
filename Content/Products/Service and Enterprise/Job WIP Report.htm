Job WIP Report

Overview

Use the Job Work In Progress report to view any jobs that were not invoiced in full at the selected date. This is useful for determining realised profit up to the selected date and determining which jobs need to be invoiced.

This report includes archived jobs that have been partially invoiced, and excludes archived jobs that have not been invoiced at all.

View the Job WIP report

To view the report:

  1. Go to Reports > View Reports > Jobs > WIP.
  2. Select the Date (Past & Including). The report will include all jobs before and inclusive of that date.
  3. Under Difference Based, select whether the difference in the report is calculated by Direct Cost or Direct Cost Inc Overhead.
  4. Select Excluded or Included to exclude or include Committed Cost in Direct Cost.
  5. Under Job Type, select to Show All Jobs, Only Project Jobs, or Only Service Jobs.
  6. Under Job Type, select to Show All Jobs, Only Project Jobs, or Only Service Jobs.
  7. Filter the report by selecting from the drop-down lists.
  8. Click Search.

Red values indicate negative value and green values indicate positive value.
