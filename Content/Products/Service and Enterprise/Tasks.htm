Tasks

Overview

In simPRO, you can use tasks to keep an interactive to-do list for quotes, jobs, customers, and internal needs.

A task can only be assigned to one person, and that person will receive emails and simPRO alerts for key dates and updates. If you would like additional people to receive email notifications about the task (including customers), add them as stakeholders through the task's Stakeholder tab. Stakeholders will not receive alerts in simPRO.

A customer must be associated with every task. They will not receive any emails or alerts about the task unless they are also assigned as a stakeholder, but will be able to view task details if they are using the customer portal with the Tasks menu enabled. If you wish to create an internal-only task, such as 'Conduct stocktake', create a customer card file for this purpose, such as 'Internal tasks' or your own company name.

To view all created tasks, go to Tasks > View. Alternatively, view tasks in the card file of the customer they are associated with by going to History > Tasks, or in the job or quote they were created in by going to Details > Tasks.

If you frequently carry out the same set of tasks for certain types of quotes or jobs, you may wish to use task templates to save time.

Create a task Create a task from the main menu

To create a task from the main menu:

  1. Hover over Tasks, and click New. A new window will open.
  2. Enter a Subject.
  3. Select the Billable and Show on Job Card check boxes as required.
  4. Enter a Description and Notes as desired. Notes are for internal use only.
  5. Under Associate with a Customer, search and select a Customer (see note below) and enter other relevant details.
  6. If desired, search and select a Project No. This will make the task visible within the selected project, and on the project's job card (if Show on Job Card is selected above).
  7. Enter a Start Date, a Due Date, or both. The Start Date is when the person assigned will first receive an alert; if no date is selected, no alerts will be sent.
  8. If desired, enter the Estimated time required.
  9. Under Internals, select a person the task will be Assigned To. A task can only be assigned to one person.
  10. Select a Priority, Status, and Category as desired. The default priority is 'Medium' and the default status is 'Pending'.
  11. Click the Stakeholders tab.
  12. Find the people you wish to add as stakeholders, and click their names to assign them to the task. Repeat as necessary. When you assign a task to another person, you are automatically added as a stakeholder.
  13. If desired, add attachments through the Attachments tab.
  14. Click Save or Finish.

Each task must be associated with a customer. If you wish to create an internal task, create a customer for this purpose, such as 'Internal Tasks' or simply use your own company name.

 Create a task from within a quote, job, or customer card file

To create a task from within a quote, job, or customer card file:

  1. Open the quote, job or customer card file, or create a new one.
  2. For a quote or job, go to Details > Tasks; for a customer card file, go to History > Tasks.
  3. Click Create Task.
  4. Follow the rest of the creation process above from step 2.

Creating a task from within a quote or job will automatically populate the task's Associate with a Customer fields with the relevant customer information. You can manually edit the customer details if desired.

 Add notes or dependencies to a task

After clicking Save in a new task, or opening an existing task, the Notes and Dependencies tabs will be visible.

Add a note to a task

If a task is particularly complex, you may wish to create a follow-up note as a reminder for activities or actions that still need to be finalised.

To add a note to a task:

  1. Open a task, or create a new one and click Save.
  2. Click the Notes tab.
  3. Click Create Note.
  4. Enter a Subject and Note as desired.
  5. Select a Follow Up date if required.
  6. Select a person from the Assign To drop-down list if required.
  7. Click Save.

Click a note to edit it, or click the red delete icon then click OK to delete notes as required.

 Add a dependency to a task

Dependencies are sub-tasks; tasks that should be completed to complete the original task. For example, if a task is called 'Get council approval', the sub-tasks may be 'Call council', 'Arrange council visit', 'Submit council form' and so on.

Dependencies may be useful when teaching someone a standard process. If you just want to explain the process of a task without creating separate dependencies, simply list the details under Details > Description. Dependencies do not need to be completed for the original task to be marked as complete.

It is best practice to keep your processes streamlined. Dependencies can be useful, but you may wish to be cautious of over-administration.

To add a dependency to a task:

  1. Open a task, or create a new one and click Save.
  2. Click the Dependencies tab.
  3. If desired, select a different Parent Task from the drop-down list. The new dependency will be listed under the parent task's Dependencies tab.
  4. Click Create Dependency. A new window will open.
  5. Enter the dependent task's details.
  6. Click Finish. The window will close.
  7. Click Save or Finish.
 Edit a task

Edit a task to change any of its key details, including its Priority, Status or Category. This can also be done using the bulk-manage options.

To edit a task:

  1. Go to Tasks > View, or another task list as desired.
  2. On the task you wish to edit, click Options > Delete Task, or simply click the task.
  3. Make changes as desired.
  4. Click Save or Finish.
 Delete a task

To delete a single task:

  1. Go to Tasks > View, or another task list as desired.
  2. On the task you wish to delete, click Options > Delete Task.
  3. Click OK.

To delete multiple tasks at once:

  1. Go to Tasks > View, or another task list as desired.
  2. Select the check boxes of the tasks you wish to delete, or select Select All below the tasks list.
  3. Click Delete below the tasks list.
  4. Click OK.
 Complete a task assigned to you View alerts for tasks assigned to you

When a task is assigned to you, you will receive an email summarising the new task. When the task reaches its start date or becomes overdue, you will receive an alert in simPRO. Alerts only appear to the person the task is assigned to, and will not appear for stakeholders.

To view simPRO task alerts for tasks assigned to you:

  1. Click the red Alerts icon at the top of the screen.
  2. If a task alert is visible under the Tasks section, click its task number to open and view it.

Read the task Subject, Description and Notes for the details you need to complete the task, and view any relevant Attachments and Dependencies. Save additional Notes if required.

 Record the actual time a task took to complete

Recording the actual time taken to complete tasks is most useful for billable tasks, where it will carry over if the task is used to create a job. Recording time is also helpful for monitoring productivity and planning future work.

You can use the built-in timer to record time as you complete a task, or manually enter the actual time afterwards.

To use the built-in timer in a task:

  1. Open the task or create a new one.
  2. Click the Details tab.
  3. Next to Actual Time, click Start to start the timer.
  4. When the task is complete, click Stop.
  5. Click Save or Finish.

The time recorded by the timer is cumulative, so if you leave the task and come back and use the timer again, it will add to the time already recorded.

To manually record the actual time a task took to complete:

  1. Open the task or create a new one.
  2. Click the Details tab.
  3. Under Actual Time, enter the hours, minutes and seconds, as required, the job took to complete.
  4. Click Save or Finish.
 Mark a task as closed

Once you have completed a task, you need to mark its status as 'Closed'.

To mark a task as closed:

  1. Open the task and go to the Details tab.
  2. Under Internals, select Closed / Closed Ahead / Closed Behind from the Status drop-down list as applicable.
  3. Click Save or Finish.

You and any assigned stakeholders will receive an email notifying you that the task has been closed. The task will no longer be listed under Open tasks in Tasks > View.

If you have a number of tasks to mark as complete, you may wish to use the bulk-manage options.

 Advanced Filter tasks

Use a range of filters to narrow down the tasks displayed. For example, you may wish to view only tasks assigned to or created by a certain person.

To filter tasks:

  1. Go to Tasks > View.
  2. Next to the search bar, click Advanced.
  3. Select an option from the desired drop-down lists under Filter by, such as Status.

The task list will update accordingly. You can then click the download icon in the top right of the table to export the filtered results, or save the filter options as a saved search to access a particular task list more quickly.

When you filter tasks by Stakeholder, you can only view stakeholders who are employees or contractors.

 Modify the task table view

You can modify the Tasks table view to include or exclude information about a task, as well as change the order of the columns the information is presented in. You can then view the fields of information you include when you click the download icon in the top right to export the Tasks table, as well as when you access tasks in jobs and quotes.

To add, remove, or reorder fields in the Tasks table:

  1. Hover over Tasks, then click View Tasks.
  2. Click Modify Table View.
  3. Select the fields you wish to include in the Tasks table.
  4. Clear check boxes for fields you wish to exclude. You cannot exclude the Order and Task No. fields.
  5. Click and drag the Reorder icon on fields to change the order of their columns in the Tasks table.
  6. You cannot reorder the Show Descriptions or Show Notes fields.
  7. Click Finish.

Modifying the table view changes the settings for all users. It is recommended that you use security groups to restrict modifying table view permissions to specific employees.

 Bulk-manage tasks

Use bulk-manage options to change multiple tasks' status, priority, and category; or to change the person they are assigned to.

To bulk-manage tasks:

  1. Go to Tasks > View, or another task list as desired.
  2. Select the check boxes for the desired tasks, or click Select All below the task list.
  3. Under With selected, select an option from each of the drop-down lists as required.
  4. Click Update.
 Create a job from a task

To create a job from a billable task:​

  1. Go to Tasks > View, or another task list as desired.
  2. Select the check box for the task you wish to convert to a job. You can select multiple tasks to create multiple jobs.
  3. Click Create Jobs.

The task will be deleted, and a new job created using the task's details.
