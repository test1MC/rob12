Set up Security Groups

Overview

The purpose of a security group is to restrict or limit access to various functions within simPRO based on a user's role in your organisation. For example, your accounts department might not necessarily require access to quotes or pre-builds, or you might wish to limit the value of quotes or purchase orders issued by a particular user or user group.

Security groups are defined in System > System Setup.

Create a security group

To create a security group:

  1. Go to System > Setup > System Setup > Security Groups.
  2. Click Create Security Group.
  3. Enter a descriptive Name.
  4. Enterprise only: Select a Business Group from the drop-down list (this will only appear if you have already set up business groups).
  5. Enterprise only: Select a dashboard view from the Dashboard drop-down list.
  6. Use tabs in the Permissions section to allow or restrict access to various functions.
  7. Click Tick All to grant access to all areas of simPRO.
  8. Click Finish.

On the System Setup > Security Group page, leave the All security group intact as you need at least one group with access to all areas.

 Enable hierarchy

If the Enable Hierarchy check box is selected in security group settings, you can rearrange security groups as per the hierarchy followed in your organisation.

This means the users can only assign a security group to an employee or a contractor card who is at or below the same security group level as theirs. They can view but not edit (it will be greyed out) the Settings > Account Setup section of an employee or a contractor card with a higher security group level than theirs. The edit option is only available when a user has access to edit employees and/or contractors.

Users will be able to apply a higher security group to another user if access to edit security groups is selected.

To enable hierarchy:

  1. Go to System > Setup > System Setup > Security Groups.
  2. Select the Enable Hierarchy check box.
  3. Use the drag handles to rearrange security groups into the desired order.

If the Enable Hierarchy check box is cleared, security groups will be arranged in alphabetical order.

To assign a security group:

  1. Go to People > Employees or Contractors.
  2. Click an employee or contractor.
  3. On the Profile tab, click Settings.
  4. Select a security group from the Security Group drop-down list.
  5. Click Finish.
 Dashboard views

The dashboard is the home screen of simPRO. If you use simPRO Enterprise, the information the dashboard displays will depend on your security group settings. The different views available are:

Service

Displays a map with quotes and jobs markers to show site locations. You can also view a list of schedules for the day and current mobile activity.

Project

Displays an activity graph with the Cost of Active Jobs and Invoiced amounts. You can also view a list of Jobs and Tasks for the current day.

Accounts

Displays a graphical representation of the current amount due on customer or site invoices with their payment terms. You can also view a list of Overdue invoices, invoices Due Today and Notes.

Alerts, Announcements and Reminders appear on every dashboard view, but information displayed in these sections will depend on security group settings.

 Advanced Separate archive and delete options

Users can be given access to the Archive Only option to prevent an accidental permanent deletion of data.

To set this up, go to System > Setup > System Setup > Security Groups > Permissions and select the Archive Only option as needed.

 Modify table view

If users have security group permission to access the Modify Table View option on most tables, they can change the table view of leads, quotes, jobs, orders, etc.

As these changes will modify the table view for every user, it is recommended to restrict most users from having Modify Table View access.

To change access, go to System > Setup > System Setup > Security Groups > Permissions and select or clear the Modify Table View option as needed.

 View employee pay rates

The View Employee Pay Rates option enables access to the Pay Rates section in the Rates tab of an employee card, a contractor card, and My Account.

To enable:

  1. Go to System > Setup > System Setup > Security Groups.
  2. Select the security group you wish to edit.
  3. Click the Setup tab.
  4. Select the View Employee Pay Rates check box ​in the Admin tab.
  5. Click Finish.
 Manage employee account setup

The Manage Employee Account Setup option enables access to the Account Setup section in the Settings tab of an employee card, a contractor card, and My Account.

To enable:

  1. Go to System > Setup > System Setup > Security Groups.
  2. Select the security group you wish to edit.
  3. Click the Setup tab.
  4. Select the Manage Employee Account Setup check box in the Admin tab.
  5. Click Finish.

Users can view but not edit the Employee Account Setup section if they have no access to this option.

 Change username and password

When Manage Employee Account Setup is disabled, users can only change their username and password through Banner Menu > User's Name > My Account, and not through the employee card.
