Tax Codes

Overview

Your taxation system will use many tax codes to ensure appropriate revenue is set aside according to your local tax authorities' requirements.

In simPRO, these codes will assist when exporting transactions to your accounting package, and ensuring your pricing incorporates the appropriate revenue when processing financial information for your customers, contractors and suppliers.

Create a new tax code

To create a new tax code:

  1. Go to System > Setup > System Setup > Accounts > Tax Codes.
  2. Click Create Tax Code.
  3. Enter the Tax Code recognised by the local tax authority for this tax.
  4. Enter the Tax Rate.
  5. Enter a Tax Name describing the code (read more on managing tax codes).
  6. If desired, select the check boxes to set the code as default for part income (all jobs and invoices), labour income (all jobs and invoices) or purchases.
  7. If you operate in the US, assign tax code members if desired.
  8. Click Create.

Please ensure your tax code has the same name as in your accounting package. For example, if it is NCG in your accounting package, create an NCG tax code in simPRO.

Select a tax type (US only)

If your business operates in the United States, you will need to set tax types for your tax codes. This allows for instances where you may need to apply several tax codes to the same invoice, such as local, state and federal taxes.

The three tax types are:

Single

This can be used as part of a combined tax code or by itself.

Component

This can only be used as part of a combined tax code.

Combined

A sum of multiple tax codes, or members, including both single and component tax codes.

Once a tax code type has been set as a component, it cannot be changed to another type.

 Assign tax code members (US only)

When you create a combined type tax code, you can assign component and single tax codes as members. The combined tax code will represent the sum total of its tax code members.

To assign tax code members:

  1. Create the individual tax codes that make up the combined tax code.
  2. Create a new tax code to represent the group, leaving the Tax Rate at the default 0 %.
  3. Enter a Tax Name that clearly describes its use.
  4. Under Available Tax Code Members, click each tax code that you wish to make a member of your new combined tax code, or alternatively, click Add All.
    • Click Assigned Tax Code Members to remove them from the combined tax code.
  5. Click Save or Create.

Once these tax settings reflect the tax setup in your accounting package, the sum total of a combined tax code that includes members is sent to the accounting package and divided into individual member tax rates upon receipt.

Because of the limit on information that can be posted to accounting packages, avoid changing the tax rate of a tax code once it has been assigned as a tax code member.

To change the tax rate of a single or component tax code that is assigned to a combined tax code, remove the tax code as a member, make changes as desired, then re-assign it to the tax code.

 Edit a tax code

To edit a tax code:

  1. Go to System > Setup > System Setup > Accounts > Tax Codes.
  2. Click the tax code you wish to edit.
  3. Edit as desired, then click Save.
 Delete a tax code

If you delete a tax code that has been used on any pending or in-progress jobs, quotes, invoices etc, you will be prompted to assign a new tax code to replace the one being deleted.

To delete a tax code:

  1. Go to System > Setup > System Setup > Accounts > Tax Codes.
  2. On the tax code you wish to delete, click the red delete icon.
  3. In the pop-up window, select another tax code from the drop-down list that will replace the deleted tax code on any pending or in-progress projects.
  4. Click OK.
 Manage changes to tax codes

When a tax code is created, it is allocated an internal ID in the simPRO database to keep track of it even if its details are modified. If you remove that tax code, you will be prompted to reassign the internal ID to a different tax code.

If your jurisdiction makes changes to these tax codes, initially make use of the tax name to describe the tax fully and include a date from which the tax rate applies. When a change comes through, create a new tax code with the new rate and a name that reflects the new start date (the tax code itself may be the same). At the changeover, archive the old and start using the new.

If this new tax code has been included as a tax member (US only) into a group, you will need to replicate the process at group level and create new groups in which the new tax code is a member.

 Adjust tax codes in quotes and jobs

You can change the tax code on each line item for labour and materials when creating a quote or job.

To change a tax code for parts or materials within a quote or job:

  1. Open a quote or job.
  2. Go to Parts & Labour > Billable.
  3. On the labour or part you wish to adjust, click Options.
  4. Click Change Tax Code.
  5. Select a Tax Code from the drop-down list.
  6. Click Save.

When your quote or job is not locked, you can also change the tax code on the cost centre. This change will overwrite your line item tax codes once finalised.

To adjust the tax code on a cost centre within a quote or job:

  1. Open a quote or job.
  2. Go to Details > Summary.
  3. Under Parts & Labour, click the pencil icon next to the tax entry.
  4. Select a Tax Code from the drop-down list and click Save.
  5. Click Save or Finish.
 Assign tax codes to an item

You can assign a purchase tax code and sales tax code to a specific catalogue item or item in stock.

To assign tax codes to an item:

  1. Open an item or create a new one.
  2. Go to Details.
  3. Select a Purchase Tax Code and / or Sales Tax Code from the drop-down list.
 Assign a tax code to a supplier

If needed, you can assign a purchase tax code to a supplier that will automatically apply to transactions with that supplier, or use the catalogue item purchase tax code.

To assign a tax code to a supplier:

  1. Open a supplier card file or create a new one.
  2. Go to Profile > Settings.
  3. Select a Purchase Tax Code from the drop-down list.

Assign tax codes to a customer

If needed, you can assign tax codes to specific customers that will automatically apply to transactions with that customer, or use the part tax code and labour tax code.

To assign tax codes to a customer:

  1. Open a customer or create a new one.
  2. Go to Rates > General.
  3. Select a Part Tax Code and / or a Labour Tax Code.
Assign tax codes to a site

If needed, you can assign tax codes to a specific site. This is useful for companies managing destination-based sales taxes, and will override tax codes assigned to customers.

To assign tax codes to a site:

  1. Open an existing site or create a new one.
  2. Go to Profile > Tax Settings.
  3. Select a Part Tax Code and / or a Labour Tax Code. If you select Use Customer Default, the tax rate assigned to the customer will be used in quotes and jobs.
 Assign a tax code to a pre-build

When creating a pre-build, you can assign a specific tax code or use the default part tax code or the default labour tax code.

To assign a tax code to a pre-build:

  1. Open an existing pre-build or create a new one.
  2. Go to Details > Sales Tax Code. Read important notes on pre-build tax codes below.
  3. Select a tax code.

Sales Tax Code: If you are working in countries with differing tax codes, such as the US, you may wish to select certain tax codes on your pre-build:

  • For materials-only pre-builds, select Use default part tax code or a specific materials tax code
  • For labour-only pre-builds, select Use default labour tax code or a specific labour tax code
  • If your pre-build has both materials and labour, select any tax code but ensure you check and update line item tax codes in each quote or job as required.
 VAT on non-GBP quotes and invoices (UK and Enterprise only)

When invoicing your customers in foreign currencies, the quote and invoice forms will display the exchange rate and tax breakdown on VAT separately. The total amount due will display in the customer's associated currency.
