Extensions

Overview

Use the System > Setup > Defaults > Extensions tab to set up mobility, tracking and system extensions in simPRO. Click Save after making changes.

Set up mobility extensions Set up eForms

Use simPRO eForms to digitise your existing paper forms for use on tablet devices in the field. It is integrated with simPRO and can also be used standalone.

To integrate eForms with simPRO after purchase:

  1. Go to System > Setup > Defaults > Extensions > Mobility Extensions.
  2. Under eForms, select On.
  3. Under eForm Attachments:
    • select Public to make eForm attachments on quotes and jobs visible in Connect and the customer portal.
    • select Email to make eForm attachments available to email with their associated quotes and jobs.
      • Quotes: the eForms attachment will be sent with the quote as an additional attachment.
      • Jobs: the eForms attachment will be sent with the invoice as an additional attachment.
      • Select the Email Only form in quotes and jobs to send an email with only the attachment.
  4. Click Save.
 Set up SMS

simPRO users in Australia and New Zealand must purchase the SMS extension from third-party provider Street Data to generate text messages in simPRO.

simPRO users in the United Kingdom need to purchase the extension from third-party provider Textlocal.

To set up SMS for simPRO:

  1. Go to Streetdata / Textlocal and follow the prompts to sign up for a Free Trial / free account.
  2. In simPRO, go to System > Setup > Defaults > Extensions > Mobility Extensions.
  3. Select the On option for SMS.
  4. Enter your new SMS Username an SMS Password.
  5. Click Save.
 Set up electronic payments

To process credit card payments over the phone and in the field and pay invoices online and through the customer portal, please see our article on simPRO Payments. This feature is enabled through third-party provider IntegraPay.

 Set up tracking extensions Set up simTRAC

Use simTRAC vehicle tracking to manage and monitor your fleet in real time. view vehicles' daily movements, and send messages to your technicians on the move. simTRAC can be integrated with simPRO or used standalone.

To integrate simTRAC with simPRO after purchase:

  1. Go to System > Setup > Defaults > Extensions > Tracking Extensions.
  2. Under simTRAC, click On.
  3. Click Save.
 Set up Navman

To integrate an existing Navman account with simPRO:

  1. Go to System > Setup > Defaults > Extensions > Tracking Extensions.
  2. Under Navman, click On.
  3. Enter a Navman Username and Navman Password.
  4. Click Save.
 Set up system extensions Set up STCs / VEECs (AU and Enterprise only)

If you deal with Small-scale Technology Certificates (STCs) under the Australian Government's Small-scale Renewable Energy Scheme, or Victorian Energy Efficiency Certificates (VEECs), you can set these up in simPRO.

To set up STCs / VEECs:

  1. Go to System > Setup > Defaults > Extensions > System Extensions.
  2. Under STCs, select On.
  3. If desired, select the Enable VEECS check box.
  4. Click the Financial sub-tab.
  5. Enter STCs Value, VEECs Value, STCs Account and VEECs Account details as applicable.
  6. Click Save.
 Set up Google Calendar integration

Enable two-way integration between users' Google Calendars and simPRO for a comprehensive view of availability.

This feature must first be enabled in both Google Apps and simPRO before integrating users' individual calendars. The ability to set up this feature in simPRO may be restricted by a user's security group settings.

This feature is only available for businesses using Google Apps for Work.

Install in Google

This process can only be done by your Google Apps for Work administrator. If you are the administrator, please follow these instructions to install simPRO Calendar Integration.

Enable in simPRO

To enable Google Calendar integration in simPRO:

  1. Go to System > Setup > Defaults > Extensions > System Extensions.
  2. Under Google Calendar Integration, select On.
  3. Click Save.

If you are using a multi-company build, this will enable Google Calendar integration for all companies in your build.
